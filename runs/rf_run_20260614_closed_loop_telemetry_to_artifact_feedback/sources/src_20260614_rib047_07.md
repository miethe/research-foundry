# Source Card: Scaling Laws for Reward Model Overoptimization

## Summary
This OpenAI paper (Gao, Schulman, Hilton; ICML 2023) empirically measures reward-model overoptimization in RLHF: optimizing too hard against an imperfect proxy reward model eventually degrades true ("gold") performance, a concrete operationalization of Goodhart's law. Using a synthetic setup where a fixed gold reward model stands in for human labelers, it fits distinct functional forms by optimization method — quadratic in distance for best-of-n sampling and logarithmic for RL — and shows the coefficients scale smoothly with reward-model parameter count, enabling a-priori budgeting of safe optimization against a KL-distance guardrail.

## Key evidence
- (ev_001) Over-optimizing a proxy reward model hinders ground-truth performance per Goodhart's law.
- (ev_002) Synthetic gold-vs-proxy setup substitutes a fixed gold reward model for human labelers.
- (ev_003) Best-of-n form is quadratic in distance: R_bon(d) = d(alpha_bon - beta_bon * d).
- (ev_004) RL form is logarithmic: R_RL(d) = d(alpha_RL - beta_RL * log d) — form depends on method.
- (ev_005) Coefficients scale smoothly with reward-model parameter count in both methods.
- (ev_006) alpha_RL can be held constant across RM sizes, giving a clean scaling curve for beta_RL.
- (ev_007) Also studies dataset size, parameter counts, and the RL KL-penalty coefficient.

## Limitations
- Results come from a synthetic setup using a fixed "gold-standard" reward model as a proxy for humans, not from human preference data; functional forms are empirical fits within that regime and may not transfer unchanged to all real RLHF pipelines.
