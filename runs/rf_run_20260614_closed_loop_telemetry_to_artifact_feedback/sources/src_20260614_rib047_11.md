# Source Card: Always Valid Inference: Bringing Sequential Analysis to A/B Testing

## Summary
This methods paper (Johari, Pekelis, Walsh) shows that ordinary frequentist p-values and confidence intervals lose their error guarantees when experimenters continuously monitor an A/B test and stop adaptively (the "peeking" problem). It introduces "always valid" p-values and confidence intervals that hold valid inference at every decision point, framed as an interface to a sequential hypothesis test, with extensions to multiple-hypothesis control. The method was deployed in a large commercial A/B platform analyzing hundreds of thousands of experiments.

## Key evidence
- (ev_001) Standard p-values/CIs are unreliable under continuous monitoring / endogenous sample-size choice.
- (ev_002) Always-valid p-values and CIs provide valid inference whenever a decision is made.
- (ev_003) The construct acts as an interface to a sequential hypothesis test users can tailor.
- (ev_004) Always-valid p-values extend to multiple-hypothesis-testing control in the sequential setting.
- (ev_005) Implemented in a large-scale commercial A/B platform analyzing hundreds of thousands of experiments.
