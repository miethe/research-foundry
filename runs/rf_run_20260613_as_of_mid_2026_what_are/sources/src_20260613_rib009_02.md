# Source Card: Deep Research - GPT Researcher Documentation

## Summary
This is the official GPT Researcher documentation page for its Deep Research report type. It establishes the runtime cost/latency profile (~5 minutes, ~$0.40 per run on o3-mini at "high" reasoning effort), how the feature is invoked (report_type="deep"), and the tunable parameters that govern its recursive exploration (breadth, depth, concurrency, total_words). It also describes the underlying tree-like, async/await recursive search algorithm.

## Key evidence
- (ev_001) Each deep research run takes ~5 minutes and costs ~$0.40 using o3-mini at "high" reasoning effort.
- (ev_002) Invoked via report_type="deep" on the GPTResearcher constructor.
- (ev_003) deep_research_breadth = parallel paths per level (default 4).
- (ev_004) deep_research_depth = recursion levels to explore (default 2).
- (ev_005) deep_research_concurrency = max concurrent operations (default 4).
- (ev_006) Tree-like recursive exploration: breadth fans out queries, depth dives deeper.
- (ev_007) Concurrent processing via async/await runs research paths simultaneously.
- (ev_008) Recommended total_words for a deep report is ~2000.
