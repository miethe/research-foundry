# Source Card: Pricing — Perplexity API Docs

## Summary
Perplexity's official API pricing documentation lays out per-token and per-request costs for the Sonar model family. Token pricing ranges from $1/$1 per 1M (base Sonar) up to $3/$15 per 1M (Sonar Pro), while Sonar Deep Research adds citation-, reasoning-, and search-query-token charges on top of its $2/$8 base. A separate per-request search-context fee (tiered low/medium/high) applies to Sonar, Sonar Pro, and Sonar Reasoning Pro, and total query cost is the sum of token costs plus that request fee.

## Key evidence
- (ev_001) Base Sonar charges $1 per 1M input tokens and $1 per 1M output tokens, the cheapest Sonar model per token.
- (ev_002) Sonar Pro costs $3 input / $15 output per 1M tokens; Sonar Reasoning Pro costs $2 input / $8 output per 1M tokens.
- (ev_003) Sonar Deep Research is $2/$8 per 1M plus $2/1M citation tokens, $3/1M reasoning tokens, and $5/1K search queries.
- (ev_004) Base Sonar per-request fees per 1000 requests are $5 (low), $8 (medium), $12 (high) search context.
- (ev_005) Sonar Pro and Sonar Reasoning Pro per-request fees per 1000 requests are $6 (low), $10 (medium), $14 (high).
- (ev_006) Total query cost = token costs + a search-context request fee applying only to Sonar, Sonar Pro, and Sonar Reasoning Pro.

## Limitations
- Vendor list prices are subject to change without notice; the page itself is dated to its access/publication window and exact citation/reasoning/search-query charges apply specifically to Sonar Deep Research, not the other models.
