# Source Card: Sonar Deep Research — Perplexity API Docs

## Summary
Perplexity's first-party API documentation for the `sonar-deep-research` model. It establishes the model as a deep-research/reasoning model that autonomously searches hundreds of sources, with a 128K-token context window, an explicit five-axis pricing schedule (input, output, citation, reasoning tokens, and search queries), and a structured response that returns citations, search_results, and a detailed usage/cost object. A worked example shows the heavy reasoning-token economics of a single comprehensive research query.

## Key evidence
- (ev_001) The sonar-deep-research model is a research model that conducts exhaustive multi-source searches and synthesizes expert-level insights into detailed reports.
- (ev_002) Sonar Deep Research has a 128K-token context length.
- (ev_003) Pricing is $2/1M input, $8/1M output, $2/1M citation, $3/1M reasoning tokens, and $5/1K search-query requests.
- (ev_004) Each search_results object exposes title, url, date, last_updated, and snippet fields.
- (ev_005) The usage object reports prompt_tokens, completion_tokens, total_tokens, citation_tokens, num_search_queries, and reasoning_tokens.
- (ev_006) A sample comprehensive query used 11,395 completion tokens, 21 search queries, and 193,947 reasoning tokens for a reported total cost of $0.816.

## Limitations
- Pricing and rate details on vendor docs are subject to change without notice; no explicit published date is visible on the page, so the "2026-01" date is approximate (access-based).
