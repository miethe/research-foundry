# Source Card: Releases - assafelovic/gpt-researcher

## Summary
The GitHub Releases page for the open-source GPT Researcher project documents a cadence of multiple minor releases per quarter through mid-2026. The current stable release as of this card is v3.5.0 (May 28, 2026), and recent releases progressively added LLM/embedding providers and retrievers (MiniMax, X/Twitter, OpenAlex, PubMed Central fixes), per-step cost tracking, inline Gemini image generation, LangSmith observability, and a Claude Code skills integration. The v3.3.7 release (Nov 2025) was a breaking LangChain v1 migration that raised the minimum Python to 3.10 and sharply reduced dependencies.

## Key evidence
- (ev_001) v3.5.0 (May 28, 2026) is the latest stable release: stack-wide bug fixes, performance, new models and retrievers.
- (ev_002) v3.4.4 (Apr 2026) added MiniMax as native LLM/embedding provider, an X/Twitter retriever, and fixed PubMed Central retrieval.
- (ev_003) v3.4.3 (Mar 2026) added per-step cost tracking and FastAPI frontend scrape-count/enhancements.
- (ev_004) v3.4.0 (Jan 2026) introduced inline Gemini image generation embedded in reports and native LangSmith tracing.
- (ev_005) v3.4.0 also added a comprehensive .claude/skills/ directory for Claude Code integration.
- (ev_006) v3.3.7 (Nov 2025) BREAKING: Python 3.10+ required (3.9 dropped) via LangChain v1; dependencies cut from 134 to 63 lines.
