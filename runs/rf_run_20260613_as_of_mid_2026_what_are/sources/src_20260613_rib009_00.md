# Source Card: assafelovic/gpt-researcher: An autonomous agent that conducts deep research on any data using any LLM providers

## Summary
GPT Researcher (by Assaf Elovic) is an open-source autonomous research agent, distributed under the Apache 2.0 license, that conducts deep research across web and local data sources and produces detailed cited reports. Its Python API centers on a `GPTResearcher` class with `conduct_research()` and `write_report()` methods, and it is LLM-agnostic via OpenAI-compatible providers (`OPENAI_BASE_URL`) plus MCP integration for hybrid web and local data. A "Deep Research" mode adds a recursive, agentic exploration workflow.

## Key evidence
- (ev_001) Positioned as the first open deep research agent for both web and local research.
- (ev_002) Shared under the Apache 2.0 license.
- (ev_003) Core Python API: GPTResearcher(query=...) then await conduct_research() then await write_report().
- (ev_004) Includes Deep Research, an advanced recursive research workflow.
- (ev_005) LLM-agnostic via OpenAI-compatible providers using OPENAI_BASE_URL.
- (ev_006) Supports MCP integration for specialized data sources (repos, databases, custom APIs).
- (ev_007) Reports export to PDF, Word, and other formats.
