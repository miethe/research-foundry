# Source Card: Introduction - GPT Researcher Documentation

## Summary
The official GPT Researcher documentation introduction page describes GPT Researcher as an autonomous agent for comprehensive online research. It establishes the system's architecture (planner agent generating research questions plus execution agents gathering information per question) and self-reported operating characteristics: roughly 3 minutes and about $0.10 per run, aggregation of 20+ web sources, reports over 2,000 words, multiple report types (research, outline, resource, lessons), and export to PDF/Word and more.

## Key evidence
- (ev_001) GPT Researcher is an autonomous agent built for comprehensive online research across a variety of tasks.
- (ev_002) An average run takes around 3 minutes and costs about $0.10.
- (ev_003) Architecture pairs a planner agent (generates research questions) with execution agents (gather relevant info per question).
- (ev_004) Aggregates over 20 web sources per research run.
- (ev_005) Generates long, detailed reports exceeding 2,000 words.
- (ev_006) Supports research, outline, resource, and lessons report types.
- (ev_007) Exports reports to PDF, Word, and more.

## Limitations
- Performance and cost figures (~3 min, ~$0.10 per run) are vendor self-reported claims from project documentation, not independently benchmarked, and may vary with model choice, query, and number of sources.
- The introduction page does not document the report_source web/local/hybrid (custom context) modes or MCP server support; those details live elsewhere in the docs/repo and were not confirmed verbatim from this page.
