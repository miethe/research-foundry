# Source Card: ARES: An Automated Evaluation Framework for Retrieval-Augmented Generation Systems

## Summary
ARES is an automated framework for evaluating retrieval-augmented generation (RAG) systems along three dimensions: context relevance, answer faithfulness, and answer relevance. It fine-tunes lightweight LM judges on synthetically generated training data, then applies prediction-powered inference (PPI) with a small set of human-annotated datapoints to statistically bound judge error. The authors report it remains accurate under domain shift and works across eight knowledge-intensive tasks (KILT, SuperGLUE, AIS) while needing only a few hundred human annotations during evaluation.

## Key evidence
- (ev_001) ARES scores RAG along context relevance, answer faithfulness, and answer relevance.
- (ev_002) ARES fine-tunes lightweight LM judges per RAG component.
- (ev_003) It uses self-generated synthetic training data plus human-annotated points for PPI to bound error.
- (ev_004) Judges remain effective across domain shifts in queries and documents.
- (ev_005) Evaluated on eight knowledge-intensive tasks (KILT, SuperGLUE, AIS) with only a few hundred human annotations.
- (ev_006) Saad-Falcon, Khattab, Potts, Zaharia; submitted 2023-11-16, revised 2024-03-31; NAACL 2024.
