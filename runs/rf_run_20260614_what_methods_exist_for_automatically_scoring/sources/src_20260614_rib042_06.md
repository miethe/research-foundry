# Source Card: Faithfulness - DeepEval / Confident AI (official docs)

## Summary
This is the official DeepEval / Confident AI documentation for the FaithfulnessMetric, a single-turn RAG evaluation metric. It defines faithfulness as the proportion of truthful claims among all claims extracted from the generator's output, where a claim is truthful if it does not contradict the retrieval context. The metric uses an LLM-as-a-judge to first extract claims and then verify their support against retrieved context, and it is designed to evaluate the RAG generator component for both end-to-end and component-level testing.

## Key evidence
- (ev_001) Faithfulness is a single-turn RAG metric using LLM-as-a-judge to detect hallucination/misinformation relative to retrieved context.
- (ev_002) Score formula: number of truthful claims divided by total number of claims.
- (ev_003) Two-step computation: extract claims from actual output via LLM, then check support against retrieved context with the same LLM.
- (ev_004) A claim is truthful if it does not contradict facts in the retrieval context (contradiction-based, not strict entailment).
- (ev_005) Required test-case fields: input query, actual_output (generator output), and retrieval_context.
- (ev_006) Usable for both single-turn end-to-end and component-level RAG testing.
