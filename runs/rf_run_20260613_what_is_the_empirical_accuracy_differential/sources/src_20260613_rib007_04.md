# Source Card: Gemini API Pricing (Gemini Developer API pricing)

## Summary
This is Google's first-party Gemini Developer API pricing documentation (last updated 2026-06-09 UTC), establishing official list prices per million tokens across the current Gemini model lineup. It documents flagship Gemini 3.1 Pro Preview at $2.00/$12.00 (<=200K) scaling to $4.00/$18.00 (>200K), the Flash family (3.5 Flash at $1.50/$9.00, 3 Flash Preview at $0.50/$3.00, 3.1 Flash-Lite at $0.25/$1.50), and prior-generation 2.5 models, plus a Batch API offering a 50% cost reduction.

## Key evidence
- (ev_001) Gemini 3.1 Pro Preview flagship pricing is $2.00/$12.00 (<=200K) and $4.00/$18.00 (>200K) per MTok.
- (ev_002) Gemini 3.5 Flash is $1.50/$9.00 per MTok (output includes thinking).
- (ev_003) Gemini 3 Flash Preview is $0.50/$3.00 per MTok.
- (ev_004) Gemini 3.1 Flash-Lite, the cheapest current-gen model, is $0.25/$1.50 per MTok.
- (ev_005) Gemini 2.5 Pro is tiered $1.25/$10.00 (<=200K) to $2.50/$15.00 (>200K), with 50% batch discount.
- (ev_006) Gemini 2.5 Flash is $0.30/$2.50 ($0.15/$1.25 batch); 2.5 Flash-Lite is $0.10/$0.40.
- (ev_007) A Batch API provides a 50% cost reduction across models.

## Limitations
- Prices are list/published values as of the page's last-updated date (2026-06-09 UTC) and change frequently; preview-model prices may change at GA. The source did not surface a verbatim launch date for Gemini 3.5 Flash, so the claimed 2026-05-19 launch is not corroborated by this card.
