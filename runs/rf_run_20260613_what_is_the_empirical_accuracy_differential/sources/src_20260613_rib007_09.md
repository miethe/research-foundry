# Source Card: LongWeave: A Long-Form Generation Benchmark Bridging Real-World Relevance and Verifiability

## Summary
LongWeave is a long-form generation benchmark that pairs real-world relevance with objective verifiability through Constraint-Verifier Evaluation (CoV-Eval), which derives queries, materials, and constraints from predefined verifiable targets. It spans seven tasks with customizable input/output lengths up to 64K/8K tokens. Evaluating 23 LLMs, the paper finds that even state-of-the-art models struggle as real-world complexity and output length increase.

## Key evidence
- (ev_001) Across 23 LLMs, even state-of-the-art models face significant challenges as complexity and output length grow.
- (ev_002) Benchmark covers seven tasks with customizable lengths up to 64K input / 8K output tokens.
- (ev_003) Introduces Constraint-Verifier Evaluation (CoV-Eval) to balance realism with verifiability.
- (ev_004) CoV-Eval defines verifiable targets first, then generates queries, materials, and constraints from them.
- (ev_005) Prior benchmarks use either hard-to-verify metrics or oversimplified synthetic setups.
- (ev_006) arXiv:2510.24345, submitted 2025-10-28, EMNLP 2025 Findings; Qwen-affiliated author team.
