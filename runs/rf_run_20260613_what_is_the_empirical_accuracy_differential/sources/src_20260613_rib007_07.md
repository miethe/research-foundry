# Source Card: The Structured Output Benchmark

## Summary
This arXiv preprint introduces a multi-source benchmark for structured-output quality in LLMs, evaluating 21 frontier and open-weight models (8B to 358B parameters) across native text, images, and audio conversations using seven metrics. Its headline finding is that model size does not predict structured-output quality: smaller open models such as Phi-4 (14B), Schematron-8B, and Qwen3.5-35B match or beat much larger closed-source models on the primary Value Accuracy metric. As a non-peer-reviewed preprint with author-defined scoring, results should be treated as primary but provisional.

## Key evidence
- (ev_001) The benchmark evaluates 21 frontier and open-weight models, ranging from 8B to 358B parameters, across three source domains and seven metrics.
- (ev_002) It is a multi-source benchmark spanning three modalities: native text, images, and audio conversations.
- (ev_003) Phi-4 (14B) scored 0.798 Value Accuracy on text, slightly above GPT-5 (0.795) and GPT-5 Mini (0.779).
- (ev_004) Schematron-8B scored 0.754, outperforming GPT-OSS 20B (0.693) despite roughly 2.5x fewer parameters.
- (ev_005) The paper's explicit headline conclusion is that model size does not predict structured output quality.
- (ev_006) Value Accuracy, the primary metric, measures the fraction of ground-truth leaf paths where the predicted value exactly matches.
- (ev_007) The smaller open model Qwen3.5-35B scored 0.801 Value Accuracy on text, exceeding closed-source frontier models on the primary metric.

## Limitations
- arXiv preprint (v1), not yet peer-reviewed; benchmark design and scoring are defined by the authors, so reported scores are self-reported. The Qwen3.5-35B figure (ev_007) is read from a results table and is recorded without a short verbatim quote.
