# Source Card: assafelovic/gpt-researcher (GitHub README)

## Summary
GPT Researcher is an open-source autonomous "deep research" agent that produces cited, multi-source research reports for web and local data. Its README describes a planner/execution/publisher agent architecture, a LangGraph + AG2 multi-agent assistant inspired by the STORM paper, and a recursive Deep Research mode. It is Apache 2 licensed and LLM-agnostic via OpenAI-compatible APIs, with LangSmith observability.

## Key evidence
- (ev_001) Core architecture: planner agents generate questions, execution agents gather information, publisher aggregates into a comprehensive report.
- (ev_002) Pipeline steps: task-specific agent, objective questions, crawler agent per question, summarize with source-tracking, filter/aggregate into final report.
- (ev_003) Multi-agent assistant built with LangGraph and AG2, inspired by the STORM paper, running from planning to publication.
- (ev_004) Aggregates over 20 sources, generates reports exceeding 2,000 words; average multi-agent run = 5-6 page report in PDF/Docx/Markdown.
- (ev_005) Deep Research mode = recursive tree-like exploration with configurable depth/breadth, ~5 min and ~$0.4 per run on o3-mini high reasoning; hybrid Tavily + MCP retrieval.
- (ev_006) Apache 2 license; LLM-agnostic via OpenAI-compatible APIs (OPENAI_BASE_URL); LangSmith tracing/observability.
