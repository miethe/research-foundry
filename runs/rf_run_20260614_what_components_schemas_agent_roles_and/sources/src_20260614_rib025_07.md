# Source Card: PaperQA2: Superhuman scientific literature search (FutureHouse research announcement)

## Summary
This FutureHouse research announcement introduces PaperQA2, described as the first AI agent to achieve superhuman performance on scientific literature search tasks, reportedly exceeding PhD- and postdoc-level biology researchers on the LitQA2 benchmark (part of the LAB-Bench eval set). It frames three agent applications built on the same system — PaperQA2 (question answering), WikiCrow (Wikipedia-style summarization), and ContraCrow (contradiction detection) — and reports an average of 2.34 contradicted statements per paper. It was published September 11, 2024 alongside a paper and open-source code.

## Key evidence
- (ev_001) FutureHouse describes PaperQA2 as the first AI agent to reach superhuman performance across scientific literature search tasks.
- (ev_002) PaperQA2 is reported to exceed PhD/postdoc biology researchers on the LitQA2 benchmark, a piece of the LAB-Bench eval set released in summer 2024.
- (ev_003) ContraCrow finds an average of 2.34 contradicted statements per paper in a random subset of biology papers.
- (ev_004) The system spans three agent applications: PaperQA2 (QA), WikiCrow (summarization), and ContraCrow (contradiction detection).
- (ev_005) Published September 11, 2024 with a paper and open-source code release.

## Limitations
- First-party promotional announcement; the superhuman/accuracy and contradiction figures are self-reported by the developing lab and are not independently replicated within this source.
