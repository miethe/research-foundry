# Source Card: BAAI/bge-reranker-v2-m3 — Hugging Face Model Card

## Summary
The official BAAI model card for bge-reranker-v2-m3 documents a cross-encoder reranker built on bge-m3 (~0.6B parameters) that takes a query and document as input and directly outputs a similarity score rather than an embedding. The card positions it as a lightweight, easy-to-deploy, fast-inference model with strong multilingual capability, and recommends it specifically for multilingual reranking scenarios. Raw scores can be mapped to [0,1] via sigmoid, and use_fp16 trades a slight quality loss for faster computation.

## Key evidence
- (ev_001) Lightweight reranker on bge-m3 with strong multilingual capability, easy deployment, fast inference.
- (ev_002) Reranker takes query+document and outputs similarity directly, unlike an embedding model.
- (ev_003) Raw score maps to [0,1] via sigmoid.
- (ev_004) Card lists base model as bge-m3 and size as 0.6B params.
- (ev_005) Authors recommend bge-reranker-v2-m3 for multilingual reranking.
- (ev_006) use_fp16=True speeds computation with slight performance degradation.

## Limitations
- Author-published model card: capability claims ("strong multilingual", "fast inference") are author-stated and not independently verified within this source. Parameter count is given as 0.6B here, while some secondary sources cite a 278-568M range; benchmark tables (BEIR/MIRACL reranking top-100) are described qualitatively rather than reproduced with full numbers in this card.
