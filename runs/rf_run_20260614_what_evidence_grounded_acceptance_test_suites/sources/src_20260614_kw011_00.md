# Source Card: AmiGo: Computational Design of Amigurumi Crochet Patterns

## Summary
AmiGo (Edelstein, Peleg, Itzhaky, Ben-Chen; ACM SCF '22) is a primary peer-reviewed source presenting an automatic pipeline that turns a closed 3D triangle mesh plus a single seed point and stitch width into human-readable amigurumi crochet instructions using only single crochet (sc), increase (inc), and decrease (dec). It builds an intermediate Crochet Graph whose rows are geodesic-distance isolines from the seed, proves crochetability via a "coupling" condition on consecutive rows (Observation 2.3), and translates the graph to instructions with a finite-state transducer. Branching shapes are auto-decomposed into half-sphere/cylinder segments sliced at saddle isolines and joined sewing-free via join-as-you-go, ordered by a segment DAG.

## Key evidence
- (ev_001) Inputs are exactly a closed triangle mesh M, a single seed vertex s, and a stitch width w; outputs use only sc, inc(x), dec(x).
- (ev_002) Coupling theorem (Observation 2.3) is the formal crochetability criterion: coupled consecutive rows guarantee valid sc/inc/dec instructions.
- (ev_003) Rows are isolines of the geodesic distance f(v)=d(v,s) from the seed.
- (ev_004) A finite-state transducer emits dec(x)/inc(x)/sc based on the inc/dec transition (degree) rule between consecutive rows.
- (ev_005) Single-piece validity requires f with no saddles; otherwise the model is auto-decomposed into segments sliced along saddle isolines by increasing geodesic distance.
- (ev_006) Adjacent segments are joined sewing-free via join-as-you-go; segment order comes from a topological sort of the segment DAG Gsigma.
- (ev_007) Short rows are deliberately avoided to keep patterns beginner-crochetable.
- (ev_008) Reported patterns span ~586-3670 stitches and ~30-60 rows (Table 2).

## Limitations
- Method handles only closed (stuffable) surfaces; segmentation can yield thin/hard-to-crochet segments; low-resolution patterns are not guaranteed symmetric; a single seed limits user control over knit direction. The coupling/crochetability theorem is numbered "Observation 2.3" in the published text (the source prompt referenced it as 3.3).
