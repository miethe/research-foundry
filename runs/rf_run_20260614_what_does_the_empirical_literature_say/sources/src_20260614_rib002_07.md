# Source Card: Measuring Attribution in Natural Language Generation Models

## Summary
This paper introduces the Attributable to Identified Sources (AIS) framework, the canonical human-evaluation standard for measuring whether a natural language generation system's output is supported by identified source material. AIS defines attribution via an intuitive test - whether a generic hearer would affirm "According to P, s" - and operationalizes it through a two-stage human-annotation pipeline (interpretability, then attribution), using explicatures to fix sentence meaning in context. The authors (Google Research) empirically validate AIS across conversational QA, summarization, and table-to-text generation.

## Key evidence
- (ev_001) AIS formally defines (s,t) as attributable to sources P iff a generic hearer would affirm "According to P, s", with s interpreted relative to time t.
- (ev_002) AIS is presented as a new evaluation framework for assessing whether NLG output about the external world is supported by sources.
- (ev_003) Stage 1 of the annotation task judges interpretability of the output without showing the source.
- (ev_004) Stage 2 shows the source P and asks whether all of the output's information is supported by P.
- (ev_005) AIS was empirically validated via human studies across three tasks: two conversational QA datasets, a summarization dataset, and a table-to-text dataset.
- (ev_006) AIS resolves context-dependence using explicatures - paraphrases interpretable in a linguistically empty context - prior to attribution judgment.

## Limitations
- AIS is a gold human-evaluation standard, not an automatic detector; throughput and cost scale with human annotation effort. The arXiv preprint dates to Dec 2021 (v2 Aug 2022); the peer-reviewed version appeared in Computational Linguistics Vol. 49 No. 4 (Dec 2023).
