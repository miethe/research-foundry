# Source Card: FacTool: Factuality Detection in Generative AI -- A Tool Augmented Framework

## Summary
FacTool (Chern et al., July 2023; GAIR-NLP / SJTU / CMU) is a task- and domain-agnostic framework for detecting factual errors in text produced by large language models such as ChatGPT. It is motivated by three challenges of generative outputs: more tasks risk factual errors, long generations lack a clear granularity for individual facts (requiring claim decomposition), and explicit verification evidence is scarce. The authors demonstrate the approach across four tasks (knowledge-based QA, code generation, mathematical reasoning, scientific literature review) and release open-source code with a ChatGPT plugin interface.

## Key evidence
- (ev_001) FacTool is a task- and domain-agnostic framework for detecting factual errors in LLM-generated text.
- (ev_002) Long LLM generations lack a clearly defined granularity for individual facts, motivating claim decomposition.
- (ev_003) Evaluated across four tasks: knowledge-based QA, code generation, mathematical reasoning, and scientific literature review.
- (ev_004) Targets the scarcity of explicit evidence available during fact checking via tool augmentation.
- (ev_005) Released as open-source code (GAIR-NLP/factool) with a ChatGPT plugin interface.
- (ev_006) Generative models expand the range of tasks now at increasing risk of factual errors.
