# Source Card: RARR: Researching and Revising What Language Models Say, Using Language Models

## Summary
RARR (Retrofit Attribution using Research and Revision), an ACL 2023 long paper from Google Research, CMU, and UC Irvine, proposes a model-agnostic, post-hoc system that retrofits attribution onto the output of any text generation model. It operates at the claim level via a Research stage (comprehensive question generation plus web search for evidence) and a Revision stage (an agreement gate followed by a minimal-edit model). Empirically, RARR significantly improves attribution while preserving the original output far more than prior edit models, and requires only a handful of training examples, a large LM, and standard web search.

## Key evidence
- (ev_001) RARR automatically finds attribution for any text generation model's output and post-edits to fix unsupported content while preserving the original.
- (ev_002) Across several SOTA LMs and diverse tasks, RARR significantly improves attribution while preserving the original far more than prior edit models.
- (ev_003) The Research stage uses comprehensive question generation (CQGen) producing questions covering all aspects of the passage.
- (ev_004) For each query, RARR uses Google Search to retrieve K=5 relevance-ranked web pages as evidence.
- (ev_005) An agreement model checks whether passage and evidence agree; the edit model runs only on disagreement and minimally alters the passage.
- (ev_006) RARR is post-hoc and model-agnostic, requiring only a handful of training examples, a large LM, and standard web search — no retraining of the generator.
