# Source Card: CrochetPARADE (Crochet PAttern Renderer, Analyzer, and DEbugger)

## Summary
CrochetPARADE is an open-source (GPLv3) platform by Svetlin Tassev that defines an unambiguous custom grammar for crochet stitches and patterns, parses and checks any supplied pattern for correctness, then builds and renders a 3D virtual model -- making compilation itself the executable-validity gate. It adds a "debug" analyzer that flags overly loose or tight stitches (a stitch-tension/quality proxy) and exports standard-symbol 2D charts, labeled SVGs, and Blender-importable 3D files, with all computation performed locally in the browser. This makes the repo a credible self-hostable validator/scoring backend for generated crochet programs.

## Key evidence
- (ev_001) Custom grammar lets users define stitches/patterns unambiguously vs. plain English.
- (ev_002) Parser checks correctness, builds a virtual model, and renders in 3D -- compilation is the validity gate.
- (ev_003) Analyzer flags overly loose/tight stitches as a tension/quality proxy before crocheting.
- (ev_004) Exports standard-symbol charts, labeled SVGs (stitch type/row/position), and Blender 3D files.
- (ev_005) Interactive 3D: rotate/zoom/pan plus animation of the pattern-creation process.
- (ev_006) Runs fully client-side with all calculations local -- self-hostable.
- (ev_007) Open source under GPLv3, freely reusable.
- (ev_008) Author anticipates AI learning to write correct crochet instructions via the grammar + renderer.

## Limitations
