# Source Card: Learn about sensitivity labels (Microsoft Purview Information Protection)

## Summary
Microsoft Purview Information Protection sensitivity labels are persistent, clear-text metadata stamps that classify and protect content wherever it travels, enforcing encryption, content marking, container access, and default sharing-link scope. Labels follow a customizable, ordered taxonomy (example: Personal, Public, General, Confidential, Highly Confidential) where list order sets priority, only one label applies per item, and a two-tier sublabel/label-group hierarchy is supported. Labeling can be discretionary or mandatory, downgrades require justification, and M365 Copilot/agents honor the highest-priority label and release data only to users holding the EXTRACT usage right.

## Key evidence
- (ev_001) Recommended example taxonomy is the customizable five-tier set Personal, Public, General, Confidential, Highly Confidential.
- (ev_002) Ordinality is set by list order (most restrictive at bottom, least at top); only one label per item.
- (ev_003) Two-tier sublabel/label-group hierarchy denotes overall sensitivity while carrying distinct settings.
- (ev_004) Mandatory labeling forces a label before saving files, sending email/meeting invites, or creating containers; default labels supported.
- (ev_005) Copilot/agents return data only if the user holds the EXTRACT usage right when labels apply encryption.
- (ev_006) Labels are persistent clear-text metadata that travel with content and form the basis for policy enforcement; highest-priority label is honored.
- (ev_007) Downgrading to a lower-priority label triggers a required justification prompt.

## Limitations
- Documentation reflects the April 2026 product state and Microsoft's recommended/example taxonomy; specific category names (General/Confidential/Highly Confidential definitions) are organization-customizable and not mandated. Justification-on-downgrade applies to files, emails, and meetings but not to sublabels sharing a parent's priority, data assets, or container-protecting labels.
