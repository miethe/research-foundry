# Source Card: I have a zero data retention agreement with Anthropic. What products does it apply to?

## Summary
This first-party Anthropic Privacy Center article defines the scope of a zero data retention (ZDR) agreement. ZDR applies only to eligible Anthropic APIs, products that use a Commercial organization API key (including Claude Code accessed via the API), and Claude Code for Enterprise plans; under it Anthropic does not store inputs or outputs except as legally required or to combat misuse. It documents two carve-outs: User Safety classifier results are still retained to enforce the Usage Policy, and Covered Models require limited data retention and review for safety work.

## Key evidence
- (ev_001) ZDR applies only to eligible Anthropic APIs, Commercial-API-key products (including Claude Code via the API), and Claude Code for Enterprise plans.
- (ev_002) Under ZDR, Anthropic does not store inputs or outputs except to comply with law or combat misuse or harm.
- (ev_003) User Safety classifier results are still retained even under ZDR to enforce the Usage Policy.
- (ev_004) Users confirm ZDR status under Settings > Privacy Controls > Data retention period.
- (ev_005) Covered Models require limited data retention and review as part of safety work, separate from standard ZDR.

- Scope is commercial customers; does not apply to consumer plans (Claude Free, Pro, Max).
