# Source Card: AI Gateway now gives you access to your favorite AI models, dynamic routing and more — through just one endpoint

## Summary
This August 27, 2025 Cloudflare blog post announces an AI Gateway refresh that consolidates multi-provider AI traffic management behind a single endpoint. The headline additions are unified billing across 350+ models from 6 providers (Closed Beta), secure BYO-key storage via Cloudflare Secrets Store, Dynamic Routes for request-attribute-based routing, and DLP scanning within AI Gateway's Firewall. It establishes vendor-stated capabilities for sensitivity-aware routing and security controls relevant to an AI routing tier policy.

## Key evidence
- (ev_001) Cloudflare announced the AI Gateway refresh on 2025-08-27 with unified billing, secure key storage, dynamic routing, and DLP controls.
- (ev_002) Dynamic Routes define actions by request attributes and are invoked as the model name dynamic/<route-name>.
- (ev_003) Firewall DLP scanning lets admins select DLP profiles to block or flag; matches can fully block or alert.
- (ev_004) Standard DLP profiles (Financial Information; Social Security, Insurance, Tax and Identifier Numbers) are free with Zero Trust; custom profiles need an upgraded plan.
- (ev_005) Secrets Store secures BYO keys via a two-level AES-encrypted key hierarchy, scoped to AI Gateway only, with RBAC and audit logging.
- (ev_006) Unified billing (Closed Beta) spans 350+ models across 6 providers: Anthropic, Google, Groq, OpenAI, xAI.

## Limitations
- Vendor first-party announcement (promotional framing); unified billing and several features were Closed Beta at publication, so availability and exact behavior may have changed since 2025-08.
