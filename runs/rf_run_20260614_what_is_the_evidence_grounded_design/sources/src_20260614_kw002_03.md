# Source Card: AmiGo: Computational Design of Amigurumi Crochet Patterns

## Summary
AmiGo (Edelstein, Peleg, Itzhaky, Ben-Chen, Technion; ACM SCF 2022) presents a computational pipeline that converts a 3D input surface into machine-checkable single-crochet amigurumi instructions. Its core abstraction is the Crochet Graph G = (S, R union C), where vertices are stitch bases, row edges capture within-row connectivity, and column edges are stitch stems between rows. Curvature is encoded as vertex-degree increases/decreases, yarn continuity is guaranteed by a formal "coupling" constraint, and multi-part shapes are assembled sewing-free via join-as-you-go with a single user-set stitch width w controlling gauge.

## Key evidence
- (ev_001) Crochet Graph G = (S, R union C): vertices are stitch tops/bases indexed (i,j), column edges are stitch stems between rows, row edges are within-row connectivity.
- (ev_002) Curvature is accommodated by stitches that locally increase (inc(x)) or decrease (dec(x)) the stitch count by x.
- (ev_003) A transducer reads vertex connectivity to emit dec(x) (x vertices in row i feeding one head) or inc(x) (x vertices in row i+1 feeding one).
- (ev_004) If all consecutive row pairs are "coupled" (Def 2.2), valid instructions P(G) exist using only sc, inc(x), dec(x) (Obs 2.3) - the continuity guarantee.
- (ev_005) Join-as-you-go attaches each segment onto the last row of the previous one (no sewing); segment order is a topological sort of the segment DAG and shared edges require coupled boundary rows.
- (ev_006) Gauge is set by a single stitch width w; f and g are sampled on a 2D grid of width w, indexing vertices (f/w, g/w).
- (ev_007) For negative Gaussian curvature, column sampling is modulated by h(x)=tanh(-x/alpha)/2+1 with alpha=10.
- (ev_008) A 2x2 mean x Gaussian curvature table (Table 1) governs where shapes are directly crochetable versus needing preprocessing or sampling modification.

## Limitations
- Method targets single-crochet (sc) amigurumi using only sc/inc(x)/dec(x); the cited curvature table notes that some curvature combinations require preprocessing or segmentation rather than direct crocheting, so it is not a universal stitch-pattern solver.
