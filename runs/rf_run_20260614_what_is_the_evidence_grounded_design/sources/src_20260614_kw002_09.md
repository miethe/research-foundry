# Source Card: Representing Crochet with Stitch Meshes

## Summary
This peer-reviewed ACM SCF '20 paper (Guo, Lin, Narayanan, McCann; Carnegie Mellon) shows how crochet patterns can be represented in the "stitch mesh" paradigm: a tileable library of mesh faces, each carrying explicit yarn geometry, that connect along labeled edges. It adapts stitch meshes (Yuksel et al. 2012) to crochet by introducing a special "current loop" edge plus short-term (leading-loop/free-yarn) and long-term (loop-through-fabric) edge labels, and derives stitch faces from the Craft Yarn Council's international crochet notation. The face set covers chain, slip stitch, single crochet, decreases (sc2tog/sc3tog), an increase utility face, and turn/cap/edge utility faces, enabling both manual design and semi-automatic generation of crochet patterns from 3D triangle meshes.

## Key evidence
- (ev_001) Crochet is represented as a library of tiles where each tile contains yarn geometry and tiles connect along their edges.
- (ev_002) A special edge type captures the "current loop" - the single loop of yarn held on the crochet hook during fabrication.
- (ev_003) Short-term (orange) edges are crossed by the leading loop and free yarn; long-term (purple) edges are crossed only by loops.
- (ev_004) Four edge labels (Next/Previous, Future/Past) connect as previous-next and past-future pairs to tile a full surface.
- (ev_005) Decreases use multiple past loop edges (sc2tog two, sc3tog three); the "inc" utility face splits a future edge into two.
- (ev_006) An automatic pipeline maps a triangle mesh to quad (sc/turn), pentagon (sc2tog), and increase-triangle (inc) faces.
- (ev_007) A single stitch "aspect ratio" gauge parameter drives the remeshing process.

## Limitations
- The modeled tile set is explicitly incomplete: only chain, slip stitch, single crochet, sc2tog, and sc3tog are implemented; double/treble crochet and post stitches (e.g., front-post double crochet) are not representable. The automatic pipeline uses a single fixed stitch aspect ratio, producing shaping artifacts and gaps for varied stitch types and tensioning styles.
