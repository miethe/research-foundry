# Source Card: CrochetPARADE (Crochet PAttern Renderer, Analyzer, and DEbugger) — repository README

## Summary
The CrochetPARADE project README, authored by Svetlin Tassev, documents a browser-based platform that renders, analyzes, and "debugs" 2D and 3D crochet patterns from a custom domain-specific grammar. It establishes the licensing posture (GPLv3 for the engine, CC BY-NC-SA for the manual, public domain for the grammar itself) and the platform's evidence-grounded design features: ambiguity-free DSL parsing, correctness checking, interactive 3D visualization, loose/tight-stitch analysis, and exports to standard crochet charts, SVG, and Blender. It is a first-party primary source for the project's design intent and IP boundaries.

## Key evidence
- (ev_001) The website and all computational components are free and open source under GPLv3.
- (ev_002) The user manual including the grammar description is released under CC BY-NC-SA.
- (ev_003) The grammar itself cannot be copyrighted and is in the public domain — adoptable independently of the GPLv3 engine.
- (ev_004) A custom DSL grammar removes plain-English ambiguity; patterns are parsed and correctness-checked before model building.
- (ev_005) The grammar follows rules similar to those of real programming languages.
- (ev_006) Supports interactive 3D rendering, loose/tight-stitch analysis, and export to standard charts, SVG, and Blender 3D files.
- (ev_007) All computation is local/on-device with no data transmission, at the cost of slowness on old hardware for very large patterns.
- (ev_008) Hosted on Codeberg (canonical) and mirrored on GitHub; showcase assets timestamped 2025-02-22.
