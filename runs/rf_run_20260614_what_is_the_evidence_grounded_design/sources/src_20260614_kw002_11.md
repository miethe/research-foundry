# Source Card: CrochetPARADE source repository (Crochet PAttern Renderer, Analyzer, and DEbugger)

## Summary
CrochetPARADE is a free, open-source (GPLv3) platform by Svetlin Tassev that lets users author crochet patterns in a custom precise grammar, parses and validates them, builds a virtual model, and renders it in 3D for both 2D and 3D crochet projects. After rendering, users "debug" the model: the tool flags overly loose or tight stitches so they can be fixed before crocheting, reducing the need for blocking. It runs entirely client-side (no server data collection), exports standard-symbol charts, labeled SVG diagrams, and Blender-importable 3D files, and is built on SVG.js and three.js.

- (ev_001) Acronym, author (Svetlin Tassev), and 2D/3D crochet pattern scope.
- (ev_002) Parses and checks patterns for correctness, builds a virtual model, renders in 3D.

- (ev_005) Exports standard-symbol charts and labeled SVG (stitch type, row, position).
- (ev_006) All calculations run locally; no data sent to a central server.
- (ev_007) Free/open source under GPLv3; canonical repo on Codeberg (GitHub mirror).
- (ev_008) Built on SVG.js and three.js libraries.
