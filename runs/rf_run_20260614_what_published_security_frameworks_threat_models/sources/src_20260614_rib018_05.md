# Source Card: Multi-Agentic System Threat Modelling Guide (OWASP GenAI Security Project - ASI) v1.0

## Summary
This is the OWASP GenAI Security Project's Agentic Security Initiative (ASI) Multi-Agentic System (MAS) Threat Modelling Guide, Version 1.0, released April 22, 2025. Rather than introducing a new taxonomy, it applies the OWASP "Agentic AI - Threats and Mitigations" master agentic threat taxonomy to real-world multi-agent systems, using the MAESTRO (Multi-Agent Environment, Security, Threat, Risk, and Outcome) layered framework - originating from a Cloud Security Alliance blog - as the structured threat-modeling companion. It demonstrates how ASI threats such as Tool Misuse, Intent Manipulation, and Privilege Compromise manifest in MAS, including delegated/cross-agent misuse, identity sprawl, and identity spoofing/impersonation.

## Key evidence
- (ev_001) Version 1.0 of the OWASP MAS Threat Modelling Guide was released April 22, 2025 by the OWASP GenAI Security Project / Agentic Security Initiative.
- (ev_002) The guide builds on OWASP "Agentic AI - Threats and Mitigations" (the master agentic threat taxonomy) by applying it to multi-agent systems.
- (ev_003) Its objective is to apply MAESTRO as a layered, architectural companion methodology to the OWASP ASI threat taxonomy.
- (ev_004) It focuses on OWASP-defined agentic threats including Tool Misuse, Intent Manipulation, and Privilege Compromise as they manifest in MAS.
- (ev_005) Identity sprawl and access complexity is flagged as a novel MAS risk driven by the large number of interacting agents.
- (ev_006) Identity Spoofing is a core MAS threat where adversaries impersonate agents to inject false data or hijack decision-making.
- (ev_007) MAESTRO maps threats across architectural layers (foundation models to agent ecosystem) plus cross-layer threats, and references the CSA MAESTRO blog as its origin.
- (ev_008) The guide extends the ASI "Agentic AI - Threats and Mitigations" document and is meant to be used in tandem with it and the OWASP Top 10 for LLM Applications.
