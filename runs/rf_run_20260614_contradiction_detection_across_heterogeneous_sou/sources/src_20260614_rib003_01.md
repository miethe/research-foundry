# Source Card: AlignScore: Evaluating Factual Consistency with a Unified Alignment Function

## Summary
AlignScore (ACL 2023, Zha et al.) proposes a holistic factual-consistency metric built on a single "information alignment" function between two arbitrary text pieces, trained over 4.7M examples spanning seven NLP task types. A 355M-parameter RoBERTa-large variant matches or beats much larger ChatGPT/GPT-4-based metrics across 22 benchmark datasets, 19 of them unseen during training. It is offered in two sizes — AlignScore-base (RoBERTa-base, 125M) and AlignScore-large (RoBERTa-large, 355M).

## Key evidence
- (ev_001) Metric is a general information-alignment function between two arbitrary text pieces.
- (ev_002) Trained on 4.7M examples from 7 task types (NLI, QA, paraphrasing, fact verification, IR, semantic similarity, summarization).
- (ev_003) Evaluated on 22 datasets; 19 were never seen during alignment training (zero-shot).
- (ev_004) 355M-parameter model matches/outperforms ChatGPT- and GPT-4-based metrics that are orders of magnitude larger.
- (ev_005) Two sizes: AlignScore-base (RoBERTa-base, 125M) and AlignScore-large (RoBERTa-large, 355M).
- (ev_006) Model has 3 heads (3-way, binary, regression); inference uses the ALIGNED probability from the 3-way classification head, which ablations show beats the binary and regression heads.
- (ev_007) Benchmarks include TRUE (11 datasets) and SummaC (6 summarization datasets), plus other recent testbeds.

## Limitations
- Caution on a common misstatement: although the alignment model includes a regression head, the production AlignScore metric computes consistency from the 3-way classification head's ALIGNED probability (per ablation), not from a continuous regression output — so it is not accurately described as "a continuous score instead of 3-way NLI labels."
- Results are 2023-era; comparisons to "ChatGPT and GPT-4" reflect mid-2023 model versions and prompting, which have since changed.
- Due to resource constraints only the first 500k samples per dataset were used (4.7M total), so training is capped rather than exhaustive.
