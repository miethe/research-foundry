# Source Card: evilmartians/lefthook — A Git hooks manager for Node.js, Ruby, Python and many other types of projects

## Summary
Lefthook is the official Evil Martians Git hooks manager, distributed as a single dependency-free binary written in Go (codebase ~86% Go) that works across Node.js, Ruby, Python, and many other project types. Hook commands are declared in a committed configuration file, support parallel execution and glob/regexp file filtering, and can be invoked directly (e.g. `lefthook run pre-commit`) or run inside Docker via a runner. As of the accessed date the latest release is v2.1.9, dated May 29, 2026, atop 217 prior releases.
