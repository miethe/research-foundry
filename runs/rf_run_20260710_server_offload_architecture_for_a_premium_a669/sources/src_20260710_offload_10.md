# Source Card: Gemini Developer API Pricing

## Summary
The first-party Google Gemini Developer API pricing page lists per-model input/output token rates across a free development tier and a paid tier, with a parallel batch (queued) tier priced at roughly half the interactive standard rate. Rates as accessed span cheap Flash-Lite ($0.10/M in) through 3.1 Pro Preview ($2.00/M in, tiered by prompt length), establishing that offloaded, batch-mode work is materially cheaper than synchronous calls.

## Key evidence
- (ev_001) 2.5 Flash-Lite: $0.10/M in, $0.40/M out; batch $0.05/$0.20.
- (ev_002) 2.5 Flash: $0.30/M in, $2.50/M out; batch $0.15/$1.25.
- (ev_003) 3.5 Flash: $1.50/M in, $9.00/M out; batch $0.75/$4.50.
- (ev_004) 3.1 Pro Preview: $2.00/$12.00 (<=200k tokens), $4.00/$18.00 (>200k).
- (ev_005) 2.5 Flash Image: $0.039/image standard, $0.0195/image batch.
- (ev_006) Free tier for dev + paid tier; batch consistently ~50% of interactive price.

## Limitations
- Pricing is volatile and point-in-time (accessed 2026-07-10); re-verify before any budget decision.
- Per-image rates for 3.1 Flash Image could not be confirmed from the page text and are omitted.
