# Source Card: Crochet Abbreviations Master List

## Summary
The Craft Yarn Council (CYC) Crochet Abbreviations Master List is the U.S. yarn industry's canonical published reference for crochet pattern abbreviations, repeat/grouping notation, measurement units, and Tunisian-crochet tokens. It explicitly states its definitions reflect U.S. terminology and provides a US/UK/Canada term-differences table documenting how identical stitch tokens map to different stitches across locales. For a pattern parser, it supplies the authoritative token vocabulary (sc, dc, hdc, tr, dtr, ch, sl st, the *2tog* decreases), the symbol grammar (*, ** , [ ], { }, ( )), and the measurement units (in, cm, m, mm, g, oz, yd) that must be recognized and normalized.

## Key evidence
- (ev_001) The CYC list is the canonical published abbreviation reference for yarn-industry designers/publishers, and reflects U.S. crochet terminology.
- (ev_002) Core stitch tokens defined: ch, sc, hdc, dc, tr, dtr, sl st.
- (ev_003) Decrease tokens defined: sc2tog, hdc2tog, dc2tog, tr2tog (work two stitches together).
- (ev_004) Repeat/grouping notation standardized: *, ** , [ ], { }, ( ) — with ( ) also meaning "work in the same stitch or space".
- (ev_005) Same token, different stitch across locales: US sc = UK dc; US dc = UK tr; US tr = UK dtr (one-step naming offset / collision).
- (ev_006) Terminology divergence beyond stitches: US "gauge" = UK/Canada "tension"; US "yo" = "yoh".
- (ev_007) Measurement abbreviations to normalize: in/", cm, m, mm, g, oz, yd.

## Limitations
- Definitions reflect U.S. crochet terminology only; the page is a human-readable table, not a machine-readable schema or formal grammar.
- No explicit page revision date is shown (site carries a ©2009–2026 copyright); treat "published_at" as approximate.
- Designers/publishers may use special, pattern-local abbreviations not on this master list (the source notes these are typically defined at the start of a book or pattern).
