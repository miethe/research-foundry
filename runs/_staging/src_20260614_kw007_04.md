# Source Card: NCD: Normal-Guided Chamfer Distance Loss for Watertight Mesh Reconstruction from Unoriented Point Clouds

## Summary
This peer-reviewed Computer Graphics Forum paper (Li, Tan, Ou, Wang; Chongqing University) proposes Normal-guided Chamfer Distance (NCD), a loss for watertight mesh reconstruction from unoriented point clouds. NCD weights the standard coordinate-based Chamfer Distance by the angle between each mesh-sampled point's (cheaply derived) normal and the vector to its corresponding input point, injecting normal/orientation information without needing fragile normal estimation on the input cloud. It claims negligible added compute over plain CD and reports superiority over state-of-the-art CD variants on benchmark datasets, motivating a curvature/normal-aware shape-distance term for downstream scoring tasks.

## Key evidence
- (ev_001) NCD adds a normal-steered weighting to Chamfer Distance based on the angle between a mesh-sampled point's normal and the vector to its corresponding input point.
- (ev_002) It weights Euclidean distances using readily available mesh-sampled normals and eliminates the need for normal estimation on the input cloud.
- (ev_003) Input-cloud normal estimation is expensive (matrix decomposition / pre-trained models); mesh-sampled normals come cheaply from the cross product of mesh vertices.
- (ev_004) NCD incurs only a negligible increase in computational complexity versus plain CD.
- (ev_005) Across multiple reconstruction models and benchmark datasets, NCD outperforms state-of-the-art CD variants.
- (ev_006) Adding a normal-consistency term to plain CD improves fidelity but costs efficiency, the trade-off NCD avoids.

## Limitations
