# Source Card: CrochetPARADE - Crochet PAttern Renderer, Analyzer, and DEbugger (Manual)

## Summary
CrochetPARADE is an open-source platform (author Svetlin Tassev, 2023) that uses a custom grammar/DSL to define crochet stitches and patterns, aiming to remove the ambiguity of plain-English instructions. It parses and validates user-provided patterns for correctness, then builds and renders an interactive 3D virtual model that flags overly loose or tight stitches (structural validation) and supports rotate/zoom/pan, animation, per-stitch hover info, and Blender export. All computation runs locally on-device with no central data collection.

## Key evidence
- (ev_001) CrochetPARADE is a platform for creating, visualizing, and analyzing both 2D and 3D crochet patterns.
- (ev_002) It uses a custom language grammar (DSL) to define stitches and stitch patterns, avoiding plain-English ambiguity.
- (ev_003) The system parses/checks patterns for correctness, then builds a virtual model rendered in 3D.
- (ev_004) It identifies overly loose or tight stitches so users can swap them before crocheting, reducing the need for blocking.
- (ev_005) Interactive 3D: rotate/zoom/pan, animate pattern creation, hover for stitch info, export to Blender.
- (ev_006) All calculations run locally on-device; no data is collected centrally or transmitted over the internet.
