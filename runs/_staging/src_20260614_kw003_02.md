# Source Card: CrochetPARADE - Crochet PAttern Renderer, Analyzer, and DEbugger (repository)

## Summary
CrochetPARADE is an open-source platform (GitHub mirror of a Codeberg-primary repo by Svetlin Tassev) that defines a custom DSL grammar for crochet stitches and patterns, replacing ambiguous plain-English instructions with a parseable, validatable language. The engine parses any user pattern for correctness, builds a virtual model, renders it in 3D, and additionally debugs structural feasibility by flagging overly loose or tight stitches. It exports standard-symbol 2D charts, annotated SVGs (stitch type / row / position), and Blender-compatible 3D models, and frames AI/natural-language pattern generation as a decoupled future concern atop the grammar and renderer.

## Key evidence
- (ev_001) Custom DSL grammar replaces ambiguous plain-English instructions, enforcing precision so patterns can be parsed and validated.
- (ev_002) Parser checks any pattern for correctness, builds a virtual model, and renders it in 3D - the executable compile/render/validate gate.
- (ev_003) Debugger identifies overly loose or tight stitches before crocheting, reducing the need for blocking (structural feasibility, not just rendering).
- (ev_004) Exports an auto-generated standard-symbol crochet chart and an SVG identifying each stitch by type, row number, and position in a row.
- (ev_005) Exports 3D models importable into Blender for further manipulation and visualization.
- (ev_006) AI/natural-language-to-pattern generation is framed as a future possibility, confirming NL-to-DSL ingestion is decoupled from the core renderer.

## Limitations
- Self-described project documentation (overview/README), not independently benchmarked; performance and accuracy claims are the authors' own.
- The AI/natural-language capability (ev_006) is stated as a likely future direction, not a shipped feature of the current renderer.
- Code is GPLv3 while the manual/grammar is Creative Commons BY-NC-SA; grammar reuse as a "Crochet IR" carries the BY-NC-SA non-commercial / share-alike constraints.
