# Source Card: Digital Crochet: Toward a Visual Language for Pattern Description

## Summary
This Onward! 2022 paper (Seitz, Rein, Lincke, Hirschfeld; Hasso Plattner Institute) proposes the first visual, domain-specific, graph-based language for representing crochet patterns, where stitches are nodes and edges encode previous/insertion/slip-stitch relations between insertion points. The graph form gives crochet an explicit intermediate representation that unifies rows and rounds as "layers," expresses increases/decreases as edge multiplicity, and enforces a crochetability ordering constraint. A prototype editor with 2D and 3D (force-directed) views was evaluated in a think-aloud study with six professional designers, who found it removed ambiguities of standard chart notation and valued the 3D view for judging shape.

## Key evidence
- (ev_001) Crochet fabric is modeled as a directed graph of stitch nodes and structural edges, an explicit IR for arbitrary topologies.
- (ev_002) Each node has exactly one outgoing "previous" edge; insertion edges point to the insertion points the current stitch is worked into.
- (ev_003) Increases = multiple incoming insertion edges to one point; decreases = a node with multiple outgoing insertion edges.
- (ev_004) Rows and rounds are unified into "layers," stored as a node property, with no special graph structure.
- (ev_005) A crochetability (valid-sequence) constraint forbids working into an insertion point that is only created in the future.
- (ev_006) The prototype editor lays out stitches in 3D automatically via a force-directed graph library (Three.js/WebGL + D3 physics).
- (ev_007) Study with six professional designers: the language removed chart ambiguities; the 3D view was valued for assessing shape.

## Limitations
- The 3D force-directed layout is a heuristic approximation, not a physically accurate yarn/stitch simulation; the authors note a more exact model would require modeling physical yarn and stitch properties.
- Evaluation is a small qualitative study (six professional designers, think-aloud), not a quantitative accuracy benchmark.
- The ACM DL landing page is gated (HTTP 403); verbatim facts here were extracted from the authors' open preprint PDF of the identical paper and corroborate the project B1 seed report.
