# Source Card: crogen: 3D Crochet Pattern Maker (GitHub)

## Summary
crogen is a self-described desktop crochet pattern maker by GitHub user kgshear: users build a crochet pattern in a Tkinter GUI by adding rows and stitch types via buttons, while a Blender-driven 3D model updates to preview the finished piece. When done, the tool generates a written pattern that reproduces the 3D design.
