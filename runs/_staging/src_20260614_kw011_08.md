# Source Card: Modeling crochet patterns with a force-directed graph layout

## Summary
This peer-reviewed open-access Eurographics Expressive/WICED proceedings paper (Greer & Mould, Carleton University, 2025) presents a method that converts a written row-by-stitch amigurumi pattern into a graph (each stitch a node; horizontal edges = sequential connections, vertical edges = working connections, extra constraint edges = non-stitch shaping) and then solves a force-directed layout, adapting Isenburg et al. 2001. The layout is a non-linear least-squares problem balancing an edge-length energy (default unit length) against a per-node curvature energy to maximize enclosed volume, the digital analog of stuffing. The authors report models accurate in shape and size to their real-world counterparts for simple amigurumi, while flagging the constant unit edge length as the primary accuracy limitation and proposing user gauge-swatch measurements as a fix.

## Key evidence
- (ev_001) Stitch = node; horizontal edges = sequential connections, vertical edges = working connections, constraint edges = non-stitch shaping.
- (ev_002) Only increase stitches are modeled as multiple nodes.
- (ev_003) Adapts Isenburg et al. [IGG01]: inflate a planar graph via per-edge lengths and curvature minimization to maximize volume.
- (ev_004) Non-linear least-squares minimizing two energies: edge-length and per-node local curvature, starting from unit length and minimal curvature.
- (ev_005) Models reported accurate in shape and size to their real-world crochet counterparts for the tested patterns.
- (ev_006) Constant edge length = 1 is an explicit limitation; user gauge swatch (10x10 rectangle) proposed for per-edge lengths.
- (ev_007) Pull-in effect recreated via one constraint edge (first-to-last node), edge length 6 by trial-and-error, curvature omitted for those nodes.
- (ev_008) Scope limited to single-crochet-like stitches in continuous rounds ("rows" used generically for rounds).

## Limitations
- Validated only on simple, beginner-level amigurumi patterns and single-crochet-like stitches in continuous rounds; not yet shown for intermediate/advanced patterns, flattened pieces, or diverse stitch types.
- Multiple key knobs (lambda = 0.65/0.9, pull-in edge length = 6) were chosen by trial-and-error or static assumption rather than derived; constant unit edge length is acknowledged as inaccurate in general. Force-directed layout did not reach convergence within minutes for the largest tested pieces.
