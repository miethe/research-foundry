# Source Card: Digital Crochet: Toward a Visual Language for Pattern Description

## Summary
This peer-reviewed ACM SIGPLAN Onward! 2022 paper (Seitz, Rein, Lincke, Hirschfeld; HPI / University of Potsdam) proposes a visual, domain-specific, graph-based language for representing crochet pattern structure, where nodes are insertion points and edges encode previous/insertion/slip-stitch relations. It demonstrates a prototype editor that builds patterns in 2D and views them in 3D, with increase/decrease semantics carried by multiple incoming/outgoing insertion edges. A think-aloud user study with six professional designers found the language and editor let designers express 2D and 3D patterns and surfaced ambiguities hidden in standard chart notation.

## Key evidence
- (ev_001) Proposes a first visual, domain-specific, graph-based language for crochet pattern representation.
- (ev_002) Prototype editor builds in 2D / views in 3D; user study shows it expresses 2D+3D patterns and removes notation ambiguities.
- (ev_003) Increase = multiple incoming insert edges; decrease = multiple outgoing insert edges (explicit connectivity semantics).
- (ev_004) Nodes are insertion points, not stitches directly -- decoupling stitch/loop identity from positional stitch counting.
- (ev_005) Editor forces linear, "as if crocheting" construction, so node creation order equals the canonical work/traversal order.
- (ev_006) Six-designer think-aloud study: all but one reproduced the pattern in 2D; four said the editor surfaced previously hidden ambiguities.

## Limitations
- Qualitative evaluation with only six professional crochet designers (recruited from a myboshi designer survey); not a large quantitative study.
- The editor is described explicitly as a prototype / proof-of-concept; 3D rendering is a force-directed-graph approximation, not a physically accurate fabric model.
