# Source Card: My Row Counter App - Features (official site)

## Summary
This is the official vendor features page for My Row Counter, a knitting/crochet row-counter app. It documents direct Ravelry integration (yarn stash import and yarn-detail auto-import), a built-in Charts tool for creating knit/crochet charts from scratch or from images with stitch symbols, broad wearable support (Apple Watch on watchOS 5+, Wear OS, Garmin via Connect IQ, Fitbit Sense/Versa 3), and a Seller Mode that calculates project costs by hourly rate and timer and displays profit. The page also references an iOS app, Android watch support, and a separate Windows/PC desktop version.

## Key evidence
- (ev_001) Direct Ravelry integration imports the user's existing yarn stash into the app.
- (ev_002) The app can search Ravelry to auto-import full yarn details instead of manual entry.
- (ev_003) The Charts tool creates a chart from scratch or by transforming an image.
- (ev_004) Chart cells are filled with stitch symbols from a list or custom symbols.
- (ev_005) An Apple Watch version supports watchOS 5 and higher.
- (ev_006) Android Wear OS watches and Fitbit Sense/Versa 3 are supported (Garmin via Connect IQ).
- (ev_007) Seller Mode auto-calculates time-spent cost from hourly rate and the project timer.
- (ev_008) Seller Mode displays the profit from a sale on the project settings page.
