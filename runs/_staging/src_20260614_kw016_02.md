# Source Card: AmiGo: Computational Design of Amigurumi Crochet Patterns

## Summary
AmiGo (Technion; ACM SCF 2022) is a computational method that converts a closed triangle mesh, a seed vertex, and a stitch width into a beginner-friendly amigurumi crochet pattern. It models the surface as a Crochet Graph whose rows are isolines of geodesic distance from the seed, samples stitches on a width-w grid, and emits instructions using only single crochet, increase, and decrease while avoiding short rows. Crochetability is enforced via curvature-aware preprocessing (Conformal Mean Curvature Flow and a tanh-based sampling-rate modulation), and branching shapes are auto-segmented and joined-as-you-go to eliminate sewing.

## Key evidence
- (ev_001) Inputs are a closed manifold triangle mesh, seed vertex, and stitch width; output is human-readable crochet instructions.
- (ev_002) Rows are isolines of geodesic distance f(v)=d(v,s); a second function g orders vertices, sampled on a 2D grid of width w.
- (ev_003) Only sc/inc/dec stitches are used and short rows are deliberately avoided for beginner accessibility.
- (ev_004) Curvature is handled via localized Conformal Mean Curvature Flow and sampling-rate modulation h(x)=tanh(-x/alpha)/2+1, alpha=10.
- (ev_005) Branching segments are attached join-as-you-go onto the previous segment's last row, requiring no sewing.
- (ev_006) Table 2 statistics: e.g., Teddy 60 rows / 6 segments / 3670 stitches / 2.5 min; instruction generation takes a few minutes on an Intel Core i7.
- (ev_007) The method only handles closed (stuffed) surfaces.

## Limitations
- Scope is restricted to closed surfaces (stuffed amigurumi); open surfaces are out of scope.
- A 2022 seminal method: while primary for its own technique, it predates later work and should be treated as tertiary when characterizing current state of the art.
