# Source Card: CrochetPARADE (Crochet PAttern Renderer, Analyzer, and DEbugger)

## Summary
CrochetPARADE is an open-source crochet tool (by Svetlin Tassev) built around a custom DSL grammar for defining stitches and patterns, designed to remove the ambiguity of plain-English instructions. It parses and validates patterns for correctness — flagging issues such as overly loose or tight stitches — then renders an interactive 3D virtual model and exports standard crochet charts (SVG, standard symbols) and 3D files (Blender-importable). It demonstrates that a validator-gated, grammar-first authoring-and-round-trip pipeline is a working, shipped artifact rather than a proposal.

## Key evidence
- (ev_001) Custom DSL grammar lets users define stitches and stitch patterns.
- (ev_002) Grammar is designed for precision, avoiding plain-English ambiguity.
- (ev_003) Patterns are parsed and checked for correctness before the virtual model is built.
- (ev_004) Validator flags overly loose/tight stitches before crocheting.
- (ev_005) Interactive 3D model supports rotate/zoom/pan, animation, and stitch highlighting/hiding.
- (ev_006) Exports auto-generated crochet charts using standard symbols, labeled by type/row/position.
- (ev_007) Exports 3D files importable into Blender.
- (ev_008) GPLv3 software; CC BY-NC-SA manual — copyleft implications for direct code reuse.
