# Source Card: Representing Crochet with Stitch Meshes

## Summary
A peer-reviewed SCF '20 paper (Guo, Lin, Narayanan, McCann; CMU Textiles Lab) showing how crochet patterns can be represented within the "stitch mesh" paradigm: a tiling where each tile carries yarn geometry and tiles connect along their edges. To handle crochet's sequential fabrication it introduces a special edge type capturing the "current loop" held on the hook, and provides a library of mesh face types modeling common crochet stitches. This yields a machine-parseable, visualizable representation that the standard abstract pictorial symbol notation does not support.

## Key evidence
- (ev_001) Crochet fabricates a 3D surface from yarn by interlacing loops formed with a special hook.
- (ev_002) Standard pictorial crochet notation does not directly show yarn layout, hindering both novices and computational design systems.
- (ev_003) Crochet patterns are represented as a library of tiles, each holding yarn geometry, connected along edges.
- (ev_004) A special "current loop" edge type captures the loop held on the hook during fabrication.
- (ev_005) A library of mesh face types models commonly-used crochet stitches.
- (ev_006) Richness is illustrated with examples including patterns generated from 3D models.
