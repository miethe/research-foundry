# Source Card: CrochetPARADE Manual (Crochet PAttern Renderer, Analyzer, and DEbugger)

## Summary
CrochetPARADE is a browser-based tool with a domain-specific language (DSL) for describing crochet patterns unambiguously, explicitly motivated by avoiding the ambiguities of plain-English instructions. Its grammar covers stitch tokens, suffix-based modifications (e.g. front/back loop), integer-multiplier repeats, and explicit attachment points/labels; from this structured IR it computes a 3D model locally and exports GLTF/STL/OBJ meshes, SVG charts using standard symbols, and DOT-like graph notation. It also supports debugging features such as stitch-tension analysis and automatic turn-direction reversal.

## Key evidence
- (ev_001) CrochetPARADE is explicitly designed as an unambiguous DSL avoiding plain-English ambiguity.
- (ev_002) Repeat syntax multiplies a stitch by an integer (e.g. 10*sc) for compact patterns.
- (ev_003) Modifications use token suffixes such as 'fl'/'bl' for front/back loop (scbl, dcfl).
- (ev_004) Explicit attachment points include direct stitch-coordinate attachment via @[2,10].
- (ev_005) Turns automatically reverse attachment direction based on even/odd count of turn directives.
- (ev_006) SVG export yields a standard-symbol crochet chart plus a connection/view diagram.
- (ev_007) All calculation runs locally with no server collection or internet transmission.
- (ev_008) Stitch-tension analysis flags overly loose/tight stitches before crocheting.
