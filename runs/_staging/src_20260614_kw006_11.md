# Source Card: Modeling Crochet Patterns with a Force-directed Graph Layout

## Summary
This Eurographics 2025 (Expressive/WICED) paper by Greer and Mould presents a tool that takes a written crochet pattern (row-by-stitch amigurumi notation) as input, translates it into a graph, and applies a force-directed graph layout to produce a 3D model matching the hand-crocheted object in shape and size. It demonstrates the pattern -> graph -> 3D forward direction needed to round-trip an intermediate representation back into a visualizer, using a heuristic layout (not ML), and is positioned to remove the costly crochet/modify/redo iteration loop for designers and beginners.

## Key evidence
- (ev_001) Method translates a written pattern into a graph and computes a force-directed layout to yield a shape/size-accurate 3D representation.
- (ev_002) Designers can adjust the pattern from the digital model without physically crocheting, removing iteration cost.
- (ev_003) Conventional design is iterative and wasteful: crochet, evaluate, undo, remake across guesswork-heavy iterations.
- (ev_004) Tool targets both professional designers and beginners, enabling preview before physical work.
- (ev_005) Input is row-by-stitch amigurumi notation; each graph node is rendered as a sphere so individual stitches are visible.
- (ev_006) 2025 Eurographics primary source (DOI 10.2312/exw.20251057, ISBN 978-3-03868-271-4), open-access CC-BY.
- (ev_007) Prior shape-based crochet tools (Nakjan et al.) are limited to symmetrical shapes such as spheres and cylinders.

## Limitations
- Application is oriented toward amigurumi; extension to clothing or other crochet styles is stated as possible but not demonstrated.
- Uses a heuristic force-directed embedding rather than a learned/ML model (illustrates the heuristic option in the reproduce-vs-heuristic-vs-ML trade space).
