# Source Card: PaddleOCR-VL: Boosting Multilingual Document Parsing via a 0.9B Ultra-Compact Vision-Language Model

## Summary
PaddleOCR-VL is a technical report introducing PaddleOCR-VL-0.9B, a sub-1B vision-language model for end-to-end document parsing that pairs a NaViT-style dynamic-resolution visual encoder with the ERNIE-4.5-0.3B language model and supports 109 languages. The authors report state-of-the-art results on OmniDocBench v1.5 (overall 92.86, ahead of MinerU2.5-1.2B's 90.67) and leading per-task metrics, demonstrating that a compact model can be competitive with much larger document VLMs. Benchmark figures are author-reported and not independently audited.

## Key evidence
- (ev_001) Core is a 0.9B VLM coupling a NaViT-style dynamic-resolution encoder with ERNIE-4.5-0.3B, supporting 109 languages.
- (ev_002) Targets document parsing; recognizes text, tables, formulas, and charts with minimal resource consumption.
- (ev_003) OmniDocBench v1.5 overall score 92.86 vs MinerU2.5-1.2B 90.67.
- (ev_004) Leading per-task metrics: Text-Edit 0.035, Formula-CDM 91.22, Table-TEDS 90.89 / TEDS-S 94.76, reading-order 0.043.
- (ev_005) Architecture: NaViT-style dynamic high-resolution encoder + lightweight ERNIE-4.5-0.3B LM for recognition and decoding efficiency.
- (ev_006) First submitted Oct 16, 2025 (v1); latest revision v4 dated Nov 25, 2025.

## Limitations
- Benchmark scores are self-reported by the authors and not independently audited. The dispatched key_points listed Formula-CDM 91.43 and an overall range of 92.56-92.86; the v4 paper text states a single overall of 92.86 and Formula-CDM 91.22, so the paper's verbatim values are used here. "Handwriting" as a distinct output category is described as a recognition capability rather than an explicitly enumerated output element in the verified text.
