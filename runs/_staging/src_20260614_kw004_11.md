# Source Card: Curvy: A Parametric Cross-section based Surface Reconstruction

## Summary
Curvy (Mathur, Khattar, Sharma; arXiv:2409.00829v1, Sep 2024) is a learning-based method that reconstructs a 3D shape point cloud from sparse planar cross-sections. It encodes each cross-section as a compact piecewise parametric polyline (degree-5 polynomial pieces, a tensor in R^6x3 per piece) using Douglas-Peucker adaptive splitting, then uses a Graph Neural Network trained in a GAN setting against a point-cloud autoencoder's latent embedding. The representation is permutation-invariant to cross-section ordering and to circular permutation of pieces, and is designed to reduce dependence on the number of cross-sections provided.

## Key evidence
- (ev_001) Curvy uses a parametric polyline + GNN to reconstruct shapes adaptively, reducing dependence on the number of input cross-sections.
- (ev_002) Each cross-section piece is a tensor in R^6x3 of degree-5 parametric polynomial coefficients in R^3.
- (ev_003) Douglas-Peucker iterative splitting retains the k highest-absolute-angle points, densifying high-curvature regions.
- (ev_004) The representation is permutation-invariant to cross-section order and to circular permutation of pieces.
- (ev_005) Training uses ShapeNet manifold meshes with sampled surface points as ground truth.
- (ev_006) Three-step pipeline: generate cross-sections -> train point-cloud autoencoder -> train GNN (GAN setting) to match the encoded vector.
- (ev_007) Positions itself against classical methods that lack object-class generalization and use complex math machinery.
