# Source Card: Mirela Ben-Chen Publications (Center for Graphics and Geometric Computing, Technion)

## Summary
This is the official Technion publications page of Mirela Ben-Chen's lab, the authoritative host of the canonical AmiGo paper PDF (mirelabc.github.io/publications/AmiGo_lores.pdf) and its arXiv link. The AmiGo entry exposes only [paper] and [arxiv] links -- no [code], supplemental, project page, or video -- whereas peer entries on the same page (e.g. PH-CPF) do carry [code] links with a replicability stamp, evidencing that AmiGo's code/data are not publicly released. The page also confirms the group's follow-up Bridges 2024 amigurumi short paper and the SIGGRAPH 2023 regularized-geodesic-distance framework underpinning the row function.

## Key evidence
- (ev_001) AmiGo is "AmiGo: Computational Design of Amigurumi Crochet Patterns" (Edelstein, Peleg, Itzhaky, Ben-Chen), SCF 2022.
- (ev_002) The AmiGo entry exposes only PDF (paper) and arXiv links; canonical PDF at mirelabc.github.io/publications/AmiGo_lores.pdf.
- (ev_003) Same-page contrast: PH-CPF carries [code] + replicability stamp + supplemental + presentation, so AmiGo's missing [code] reflects an unreleased artifact, not a page convention.
- (ev_004) Follow-up short paper "Amigurumi Crochet Patterns from Geodesic Distances" (Ben-Chen, Edelstein), Bridges 2024.
- (ev_005) "A Convex Optimization Framework for Regularized Geodesic Distances" (SIGGRAPH 2023) is the geodesic-distance machinery behind the row function.
- (ev_006) Official Technion / Center for Graphics and Geometric Computing affiliation; authoritative AmiGo (B1) artifact location.

## Limitations
- The page itself does not state a publication date for the page; per-paper venues/years are listed but the page has no global timestamp (accessed 2026-06-14).
- Absence of a [code] link is strong but indirect evidence of non-release; it confirms no public code is linked from the official lab page, not that no code exists privately.
