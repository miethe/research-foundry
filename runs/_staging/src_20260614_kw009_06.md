# Source Card: Real-Time Knit Deformation and Rendering

## Summary
This ACM TOG / SIGGRAPH 2025 paper introduces the first real-time framework that takes an animated stitch mesh and enhances it with yarn-level simulation and fiber-level rendering, using a knot-based yarn representation solved via a GPU Gauss-Newton scheme. On an Nvidia RTX 3090 it achieves 1.05 ms/frame simulation and 23.6 ms/frame rendering, reaching near-ground-truth quality while being ~120,000x faster than a path-traced fiber-level reference. It does so by optimizing only ~100K yarn segments versus 51M-61M fiber segments in the reference geometry.

## Key evidence
- (ev_001) On an RTX 3090: 1.05 ms/frame simulation, 23.6 ms/frame fiber-level rendering.
- (ev_002) Rendering is ~120,000x faster than the path-traced fiber-level reference at near-ground-truth quality.
- (ev_003) The method optimizes only ~100K yarn segments.
- (ev_004) The reference models contain 51M, 55M, and 61M fiber segments.
- (ev_005) Knit's yarn/fiber geometry and primitive count are the core real-time challenge.
- (ev_006) INFERENCE: desktop-GPU-only (no mobile benchmarks); full fiber detail is beyond mid-range-phone budget, so approximate stitch-graph meshes are the mobile-viable path.

## Limitations
- All performance numbers are on a single high-end desktop GPU (RTX 3090); the source reports no mobile or low-power benchmarks, so mobile-viability conclusions (ev_006) are inference, not stated by the authors.
- The 120,000x speedup is measured against the authors' own path-traced fiber-level reference, not an independent third-party baseline.
