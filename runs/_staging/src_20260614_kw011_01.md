# Source Card: AmiGo: Computational Design of Amigurumi Crochet Patterns

## Summary
AmiGo (Edelstein, Peleg, Itzhaky, Ben-Chen; Technion; SCF 2022) is a computational-fabrication method that converts a closed triangle mesh plus one user-specified point into crochet instructions whose knitted-and-stuffed output resembles the input geometry. The core abstraction is a Crochet Graph (geometry + connectivity) that is translated into a pattern; the shape is auto-segmented into crochetable components joined with join-as-you-go so no sewing is required. The paper establishes crochetability and shape-similarity to the input as the two acceptance axes.

## Key evidence
- (ev_001) AmiGo turns a closed triangle mesh plus a single user point into crochet instructions that resemble the input when knitted and stuffed.
- (ev_002) The pipeline constructs a Crochet Graph (geometry + connectivity) that is then translated into a crochet pattern.
- (ev_003) Crochet Graph vertices are stitch bases/tops; column edges are stitch stems and row edges encode within-row connectivity.
- (ev_004) The shape is automatically segmented into crochetable components joined with join-as-you-go, requiring no additional sewing.
- (ev_005) Join-as-you-go crochets each segment onto the previous segment's last row, eliminating a sewing step.
- (ev_006) Demonstrated across a wide variety of shapes, yielding patterns the authors call easily crochetable (crochetability + shape-similarity as acceptance axes).
