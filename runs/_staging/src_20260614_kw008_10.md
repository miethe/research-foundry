# Source Card: Filament gltfio AssetLoader.h (glTF 2.0 importer)

## Summary
This header file defines the public interface of Google Filament's gltfio AssetLoader, the official glTF 2.0 importer for the Filament real-time rendering engine. It documents that AssetLoader consumes a single blob of glTF 2.0 content (JSON .gltf or binary .glb) and produces a FilamentAsset that bundles Filament textures, vertex buffers, and index buffers — a direct glTF-to-GPU mapping. The header also delineates the upload boundary (ResourceLoader handles external buffer fetching and texture creation) and the instancing model (createInstancedAsset yields a primary asset with multiple geometry-sharing instances).

## Key evidence
- (ev_001) AssetLoader ingests a glTF 2.0 blob (JSON or GLB) and produces a FilamentAsset.
- (ev_002) A FilamentAsset is a bundle of Filament textures, vertex buffers, and index buffers.
- (ev_003) Both .glb binary and JSON-based .gltf files are accepted by createAsset.
- (ev_004) ResourceLoader defines the upload boundary; AssetLoader does not fetch buffers or create textures itself.
- (ev_005) createInstancedAsset produces a primary asset with one or more geometry-sharing instances.
- (ev_006) The header is the official upstream Filament importer interface (2019 AOSP/Google copyright).
