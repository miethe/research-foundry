# Source Card: Digital Crochet: Toward a Visual Language for Pattern Description

## Summary
This Onward!/SPLASH 2022 (ACM SIGPLAN) paper, by Seitz, Rein, Lincke, and Hirschfeld of the University of Potsdam / Hasso Plattner Institute, frames crochet pattern description as a programming-language problem. It argues that existing crochet pattern languages are ambiguous or limited in expressiveness, and proposes a first visual, domain-specific, graph-based language, realized in a prototype 2D-authoring / 3D-viewing editor and validated via a user study that found the language removes ambiguities found in standard notations.

## Key evidence
- (ev_001) Crochet remains largely manual and is barely supported by domain-specific digital tools, unlike knitting or weaving.
- (ev_002) Existing crochet tools are constrained by pattern languages that are ambiguous or limited in expressiveness, yielding incomplete or ambiguous instructions.
- (ev_003) The paper proposes a first visual, domain-specific, graph-based language for crochet pattern representation.
- (ev_004) A prototype editor realizes the language with 2D pattern authoring and 3D viewing.
- (ev_005) A user study showed the language expresses 2D and 3D patterns and removes ambiguities found in standard notations.
- (ev_006) Published at Onward!/SPLASH 2022 (ACM SIGPLAN), framing crochet representation as a programming-language design problem.
