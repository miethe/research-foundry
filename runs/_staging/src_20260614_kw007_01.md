# Source Card: AmiGo: Computational Design of Amigurumi Crochet Patterns

## Summary
AmiGo (Edelstein, Peleg, Itzhaky, Ben-Chen; Technion; SCF 2022) is a computational-fabrication method that generates human-readable crochet patterns for Amigurumi (knitted stuffed toys) directly from a closed 3D triangle mesh plus one user-specified point. It builds a "Crochet Graph" encoding stitch geometry and connectivity, auto-segments the shape into crochetable components joined with the join-as-you-go method (no sewing), and translates the graph into a pattern. The abstract claims broad applicability and "easily crochetable" output but reports no numeric fidelity metric, and similarity is asserted against the rigid input geometry rather than a deformed-state simulation.

## Key evidence
- (ev_001) AmiGo generates crochet instructions from an input 3D model, focused on Amigurumi stuffed toys.
- (ev_002) Input is a closed triangle mesh plus a single user point; the knitted-and-stuffed output is "similar to the input geometry."
- (ev_003) The core artifact is a "Crochet Graph" encoding geometry and connectivity, translated into a crochet pattern.
- (ev_004) Automatic segmentation into crochetable components joined via join-as-you-go, requiring no additional sewing.
- (ev_005) Claims applicability to a "large variety of shapes and geometries" and "easily crochetable" patterns, with no numeric fidelity score stated.
- (ev_006) 11-page, 10-figure paper published at SCF 2022, submitted 2 Nov 2022, classified Graphics (cs.GR).
