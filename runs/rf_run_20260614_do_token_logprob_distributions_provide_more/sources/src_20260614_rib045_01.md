# Source Card: On Verbalized Confidence Scores for LLMs

## Summary
This arXiv preprint (v2, May 2026) benchmarks "verbalized" confidence scores — asking an LLM to state its own confidence as output tokens — across 10 datasets, 11 LLMs, and 17 prompt methods. It finds that reliability depends heavily on prompt formulation: large models (>=70B) reach an ECE around 0.1 and the best "combo" prompt narrows average deviation from accuracy to about 7%, while small models (e.g. gemma1.1-2b) are poorly calibrated and near-random. The work argues prompt engineering can make verbalized confidence a simple, low-overhead, model-agnostic uncertainty-quantification method.

## Key evidence
- (ev_001) For LLMs >=70B parameters, verbalized-confidence ECE is around 0.1 (~10% deviation from true accuracy).
- (ev_002) The best "combo" prompt achieves ~7% average deviation from empirical accuracy for large LLMs.
- (ev_003) The smallest model (gemma1.1-2b) is poorly calibrated and almost independent of its accuracy.
- (ev_004) Calibration is not inherent to the model; it depends heavily on how confidence is elicited.
- (ev_005) Benchmark spans 10 datasets, 11 LLMs, and 17 prompt methods.
- (ev_006) Verbalized confidence can yield well-calibrated scores with certain prompt methods, despite strong prompt dependence.

## Limitations
- Results are bounded to the evaluated model set (Gemma, Llama 3, Qwen, GPT-3.5, GPT-4o) and a base-accuracy regime of roughly 0.5-0.9; generalization to newer/larger models or other task distributions is not established.
- This is an arXiv preprint (v2 dated 2026-05-05), not a venue-published paper; calibration metrics are dataset- and prompt-dependent.
