# Source Card: Use the Claude Agent SDK with your Claude plan

## Summary
This official Anthropic help-center article establishes that, effective June 15, 2026, Claude Agent SDK and `claude -p` usage no longer counts against a Claude plan's usage limits and is instead covered by a monthly Agent SDK credit that varies by plan tier ($20 Pro, $100 Max 5x, $200 Max 20x, and tiered amounts for Team and Enterprise). The credit covers SDK usage in the user's own projects, the non-interactive `claude -p` command, Claude Code GitHub Actions, and third-party apps authenticating via a subscription. API-key (Claude Platform) accounts receive no credit and continue pay-as-you-go; exhausted credits overflow to usage credits at standard API rates only if enabled, and unused credits do not roll over.

## Key evidence
- (ev_001) Effective June 15, 2026, Agent SDK and `claude -p` usage no longer counts toward plan usage limits.
- (ev_002) Monthly credit is $20 (Pro), $100 (Max 5x), $200 (Max 20x).
- (ev_003) Team Standard $20, Team Premium $100; Enterprise $20-$200 by seat type.
- (ev_004) Credit covers own projects, `claude -p`, GitHub Actions, and subscription-authenticated third-party apps.
- (ev_005) API-key (Claude Platform) accounts get no credit; pay-as-you-go continues unchanged.
- (ev_006) Exhausted credit overflows to usage credits at standard API rates, only if enabled.
- (ev_007) Unused credits don't roll over; requests stop if usage credits aren't enabled.

## Limitations
- Help-center page shows only a relative timestamp ("Over 3 weeks ago"), not an exact publish date; published_at recorded as the 2026-06 source month.
- This is a billing/policy article: it states no latency, throughput, or rate-limit guarantees for Agent SDK usage under the credit.
