# Source Card: CrochetBench: Can Vision-Language Models Move from Describing to Doing in Crochet Domain?

## Summary
CrochetBench is an arXiv (Nov 2025) benchmark that tests whether vision-language models can move beyond describing crochet imagery to actually generating executable crochet procedures. It uses the CrochetPARADE DSL as an intermediate representation so outputs can be structurally validated and functionally evaluated by execution/compilation, spanning stitch classification, instruction grounding, and NL-to-DSL / image-to-DSL translation. The headline finding is that VLM performance collapses as evaluation moves from surface similarity to executable correctness, exposing weak long-range symbolic reasoning and weak 3D-aware procedural synthesis.

## Key evidence
- (ev_001) CrochetBench evaluates the describe-to-do shift via fine-grained procedural reasoning: recognize stitches, select structurally appropriate instructions, generate compilable procedures.
- (ev_002) Adopts the CrochetPARADE DSL as intermediate representation enabling structural validation and functional evaluation via execution.
- (ev_003) Benchmark spans stitch classification, instruction grounding, and both NL-to-DSL and image-to-DSL translation.
- (ev_004) Performance sharply decreases as evaluation shifts from surface similarity to executable correctness — weak long-range symbolic reasoning and 3D-aware procedural synthesis.
- (ev_005) Frames the core gap as surface-level understanding vs. executable precision in real-world creative domains.

## Limitations
- Evidence drawn from the abstract of an arXiv preprint (accessed via fetched page); per-task quantitative scores and model list are in the full paper body and are not captured here.
