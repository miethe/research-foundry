# Source Card: Neighbor-Aware Data-Driven Relaxation of Stitch Mesh Models for Knits

## Summary
This SIGGRAPH Asia 2025 paper extends the stitch-mesh representation with neighbor-aware, per-stitch material properties so that lightweight mesh-level relaxation can capture mixed knit-purl structures rather than treating fabric as homogeneous. It derives a finite set of four-way kernels (each stitch's rest behavior conditioned on its 4 neighbors) whose rest-lengths are inferred from 11 measured basis swatches via a linear model. The resulting relaxation enables fast fabric preview and better initialization/constraints for more accurate offline yarn-level simulation, validated against physical knits.

## Key evidence
- (ev_001) Mesh-level models serve both interactive pattern editing and initialization of yarn-level sim (hybrid pipeline).
- (ev_002) Prior mesh-level methods assume homogeneous material and miss mixed structures.
- (ev_003) Core idea: stitch behavior depends on surrounding stitch types; model made neighbor-aware at stitch level.
- (ev_004) A finite set of four-way kernels combines to create general knit-purl patterns for relaxation.
- (ev_005) 4 neighbors capture much of the neighborhood-dependent deformation; fit with 11 basis swatches.
- (ev_006) Rest-lengths inferred via a linear model from measured references → fast preview + more accurate yarn-level sim.

## Limitations
- Only the author-supplied abstract was fully accessible; body/results numerics (speedups, error metrics, and the specific count of discrete kernels) were not confirmable verbatim and are omitted. The abstract states "a finite set of four-way kernels" without a verbatim numeric count.
