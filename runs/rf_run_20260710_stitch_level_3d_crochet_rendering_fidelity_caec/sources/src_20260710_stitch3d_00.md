# Source Card: Representing Crochet with Stitch Meshes

## Summary
This peer-reviewed SCF '20 paper (Guo, Lin, Narayanan, McCann) adapts the "stitch mesh" tile-based representation — originally developed for knitting — to crochet, so that patterns can be parsed, visualized, and designed computationally rather than relying on abstract pictorial symbols that omit yarn layout. Its central contribution is a special mesh edge type that captures the "current loop" (the loop held on the hook during fabrication) plus a library of mesh face types modeling common crochet stitches, demonstrated on examples including patterns derived from 3D models.

## Key evidence
- (ev_001) Crochet builds a 3D surface from yarn by interlacing hook-formed loops.
- (ev_002) Standard symbol notation omits yarn layout, hindering both novices and programs.
- (ev_003) Patterns are represented as a stitch-mesh library of tiles carrying yarn geometry, connected along edges.
- (ev_004) A special "current loop" edge type is the key adaptation enabling crochet in the stitch-mesh formalism.
- (ev_005) A library of mesh face types models common crochet stitches; patterns can be generated from 3D models.
- (ev_006) 8-page research article, Article No. 4, in the 5th Annual ACM Symposium on Computational Fabrication.
