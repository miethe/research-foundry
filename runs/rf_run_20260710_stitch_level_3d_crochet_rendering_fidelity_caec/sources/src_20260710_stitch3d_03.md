# Source Card: CrochetPARADE (Crochet PAttern Renderer, Analyzer, and DEbugger)

## Summary
CrochetPARADE is an open-source platform for creating, visualizing, and analyzing both 2D and 3D crochet patterns via a custom crochet grammar/DSL that forward-simulates written patterns into 3D models. It runs entirely client-side (all calculations local, no server transmission), establishing a concrete precedent for on-device crochet rendering. Its README documents both the local-processing design and its performance ceiling: large stitch counts and old hardware degrade responsiveness.

## Key evidence
- (ev_001) Open-source platform to create, visualize, and analyze 2D and 3D crochet patterns.
- (ev_002) All calculations run locally on-device with no server/data transmission — precedent for client-side rendering.
- (ev_003) Sluggish on old hardware; tens-of-thousands-of-stitch models take minutes or more to calculate.
- (ev_004) Code under GPLv3, manual under CC BY-NC-SA, grammar itself in the public domain.
- (ev_005) Defines a custom crochet grammar/DSL and forward-simulates patterns into 3D models (renderer + analyzer + debugger).
- (ev_006) Small self-contained codebase — 12 commits on the main branch.

## Limitations
- Performance characteristics are described qualitatively in the README, not from measured benchmarks; the exact last-commit date is not stated in the README text (source metadata dates it to Feb 2025).
