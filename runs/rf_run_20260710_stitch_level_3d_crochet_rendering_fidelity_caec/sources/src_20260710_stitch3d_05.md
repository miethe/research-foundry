# Source Card: Modeling Crochet Patterns with a Force-directed Graph Layout

## Summary
This 2025 Expressive/WICED paper by Émile Greer and David Mould (The Eurographics Association) presents a method that takes a written crochet pattern as input, translates it into a graph, and applies a force-directed graph layout to produce a 3D model matching the hand-crocheted object's shape and size. The goal is to replace the iterative physical guess/crochet/undo loop with digital pre-visualization. The application is oriented toward amigurumi but the authors note it could extend to clothing.

## Key evidence
- (ev_001) Pattern design is an iterative loop of crocheting, evaluating, and undoing/remaking stitches over multiple passes.
- (ev_002) Method: written pattern → graph → force-directed layout → 3D representation.
- (ev_003) Output 3D model matches the hand-crocheted pattern's shape and size, enabling digital adjustment without physically crocheting.
- (ev_004) Targets both professional designers and beginners for pre-visualization before physical effort.
- (ev_005) Oriented toward amigurumi; extensible to clothing or similar crochet styles.
- (ev_006) CCS classification Human-centered computing → Graph drawings; Expressive/WICED 2025; DOI 10.2312/exw.20251057.
