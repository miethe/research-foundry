# Source Card: AmiGo: Computational Design of Amigurumi Crochet Patterns

## Summary
AmiGo (Edelstein et al., ACM SCF '22, Technion) is a computational method that turns a closed 3D triangle mesh plus a single user seed point and stitch width into human-readable Amigurumi crochet instructions. It builds an intermediate "Crochet Graph" whose rows are isolines of the geodesic distance from the seed and whose square single-crochet stitches form a unit-square quad remesh, using local increase/decrease stitches to accommodate curvature. It also produces a ShapeUp-based 3D preview that closely predicts the crocheted-and-stuffed result, auto-segments branching shapes joined without sewing, and runs in a few minutes.

## Key evidence
- (ev_001) Inputs are a closed manifold triangle mesh, a single seed vertex s, and a stitch width w.
- (ev_002) Single crochet is treated as a square stitch, making mesh coverage equivalent to a constant-length unit-square quad remesh.
- (ev_003) Nonzero Gaussian curvature is accommodated by local increase (inc(x)) and decrease (dec(x)) stitches.
- (ev_004) Rows are isolines of the geodesic distance function f(v)=d(v,s) from the seed.
- (ev_005) Curvature-adapted sampling reconstructs shape fidelity where uniform sampling fails (Fig. 6).
- (ev_006) A ShapeUp 3D embedding using purely geometric conditions closely matches the physical stuffed result.
- (ev_007) Branching shapes are auto-segmented and joined via join-as-you-go, requiring no sewing.
- (ev_008) Pattern generation takes a few minutes vs tens of minutes for AutoKnit.

## Limitations
- Handles only closed surfaces (stuffed Amigurumi); does not account for model symmetry, so low-resolution patterns may crochet non-symmetrically; limited design freedom (single seed, no per-region knitting-direction control). Craters (positive Gaussian + negative mean curvature) cannot be realized and are smoothed away in preprocessing.
