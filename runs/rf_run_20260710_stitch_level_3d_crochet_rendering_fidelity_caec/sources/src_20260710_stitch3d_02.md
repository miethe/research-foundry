# Source Card: Digital Crochet: Toward a Visual Language for Pattern Description

## Summary
This ACM SIGPLAN Onward! 2022 paper by Seitz, Rein, Lincke, and Hirschfeld argues that crochet, unlike knitting and weaving, remains a purely manual craft whose existing pattern notations are ambiguous or limited in expressiveness. The authors propose the first visual, domain-specific, graph-based language for crochet pattern representation and implement a prototype editor that authors patterns in 2D and views them in 3D. A user study shows the language can express both 2D and 3D patterns while removing ambiguities found in current standard notations.

## Key evidence
- (ev_001) Crochet remains purely manual with little digital tooling, unlike knitting/weaving.
- (ev_002) Existing crochet pattern languages are ambiguous or limited in expressiveness, yielding effortful, error-prone instructions.
- (ev_003) The work proposes the first visual, domain-specific, graph-based crochet pattern language.
- (ev_004) A prototype editor creates patterns in 2D and views them in 3D.
- (ev_005) A user study shows the language expresses 2D and 3D patterns and removes notation ambiguities.
- (ev_006) Published in Onward! 2022 (ACM SIGPLAN SPLASH), pp. 48-62, 01 December 2022, open access.
