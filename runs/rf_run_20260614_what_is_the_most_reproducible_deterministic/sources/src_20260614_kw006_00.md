# Source Card: AmiGo: Computational Design of Amigurumi Crochet Patterns

## Summary
AmiGo (Edelstein et al., SCF '22, Technion) is a peer-reviewed geometry-processing method that converts a closed triangle mesh into a knit/crochet amigurumi pattern. It derives a row scalar field from geodesic distance (Heat Method), an orthogonal column field by a least-squares minimization, builds a Crochet Graph whose rows are coupled via Dynamic Time Warping, and emits sc/inc/dec stitch instructions through a transducer over coupled rows. Curvature obstructions (negative mean + positive Gaussian curvature) are detected and preprocessed away with Conformal Mean Curvature Flow.

## Key evidence
- (ev_001) Inputs are a closed triangle mesh M=(V,E), a seed vertex s, and a stitch width w; only closed surfaces are handled because amigurumi are stuffed.
- (ev_002) Rows come from the geodesic distance field f(v)=d(v,s) via the Heat Method with diffusion time t = (average edge length)^2.
- (ev_003) The column field g minimizes the integral of |<J grad f, grad g> - 1|^2 with g=0 on the longest monotone boundary path B.
- (ev_004) The Crochet Graph G=(S, R union C) requires consecutive rows to be coupled; coupling is computed by Dynamic Time Warping.
- (ev_005) A transducer emits sc, dec(x), or inc(x) from coupled-row connectivity; no explicit numeric cap on inc/dec rate is given.
- (ev_006) Negative-mean / positive-Gaussian "crater" regions cannot be realized and are preprocessed via Conformal Mean Curvature Flow until mean curvature is positive everywhere.
- (ev_007) Implemented in Matlab + C++ (Heat Geodesics, geodesic paths, ShapeUp); Teddy = 60 rows, 6 segments, 3670 stitches in 2.5 min.

## Limitations
- The paper gives NO explicit numeric cap on inc/dec rate, max stitch-count change per round, or slope limit — an underspecified stage for KnitWit's bounded-constraint goal.
- No source code or data release is stated.
- Curvature obstructions are a documented failure mode requiring mesh preprocessing before a valid pattern can be produced.
