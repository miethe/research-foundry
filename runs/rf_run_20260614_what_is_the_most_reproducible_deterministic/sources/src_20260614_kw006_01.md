# Source Card: AmiGo: Computational Design of Amigurumi Crochet Patterns

## Summary
AmiGo (ACM SCF 2022, Edelstein, Peleg, Itzhaky, Ben-Chen) is a computational pipeline that converts a closed triangle mesh plus one user-specified point into human-readable amigurumi crochet instructions. It builds the geometry and connectivity of a "Crochet Graph," automatically segments the surface into crochetable components using saddle points of a geodesic distance field, orders those segments by topological sort, and connects them with the join-as-you-go method so no sewing is required.

## Key evidence
- (ev_001) Input is a closed triangle mesh plus a single user point; output is crochet instructions approximating the geometry when crocheted and stuffed.
- (ev_002) The Crochet Graph's geometry and connectivity are constructed and then translated into a crochet pattern.
- (ev_003) Automatic segmentation into crochetable components joined via join-as-you-go, requiring no sewing.
- (ev_004) Segmentation iterates over saddle points, slicing the geodesic isoline into a new segment per connected component.
- (ev_005) Segment crocheting order is fixed by a topological sort of the directed segment adjacency graph.
- (ev_006) Method demonstrated across a large variety of shapes, yielding easily crochetable patterns.
