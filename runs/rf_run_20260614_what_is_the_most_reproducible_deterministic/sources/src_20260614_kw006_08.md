# Source Card: math_crochet - surfaces-of-revolution crochet generator (reference implementation)

## Summary
math_crochet is a small public reference implementation by GitHub user meganmartinez that generates crochet patterns for mathematical surfaces of revolution. It is written entirely in Sage and run as a worksheet in CoCalc: the user supplies a function f(x), interval bounds, stitch and row gauges, and a scale, and the program emits crochet instructions together with per-row coordinate lists and a profile-curve plot. Its scope is restricted to surfaces of revolution about the x-axis, and it is distributed under the strong-copyleft AGPL-3.0 license.

## Key evidence
- (ev_001) AGPL-3.0 license is a strong copyleft constraint relevant to embedding the code in a commercial mobile app.
- (ev_002) Codebase is 100% Sage, designed for the CoCalc environment.
- (ev_003) Tool is operated by opening a Sage worksheet in CoCalc, not as a standalone or mobile app.
- (ev_004) Inputs include f(x), bounds a/b, stitch gauge S, row gauge R, and scale; f(x) must be strictly positive on (a,b).
- (ev_005) f(x) must have a defined derivative on [a,b].
- (ev_006) Outputs are crochet instructions plus per-row coordinate lists and a plot marking each row on the profile curve.
- (ev_007) Scope is surfaces of revolution about the x-axis only (no arbitrary mesh or branching).
