# Source Card: AmiGo: Computational Design of Amigurumi Crochet Patterns

## Summary
AmiGo (Edelstein, Peleg, Itzhaky, Ben-Chen; SCF 2022, mirrored on arXiv as 2211.01178) presents an algorithm that solves the inverse problem of turning a 3D model into a human-executable amigurumi crochet pattern. From a closed triangle mesh plus a single user-specified point, it builds a Crochet Graph, automatically segments the shape into crochetable components, and joins them with the join-as-you-go method so no sewing is needed. The open-access arXiv mirror (CC BY-NC-SA 4.0, PDF + TeX source) enables reproducible citation independent of the paywalled ACM record.

## Key evidence

- (ev_004) Core mechanism is constructing a Crochet Graph that is then translated into the crochet pattern.
- (ev_005) Automatic segmentation into crochetable components joined via join-as-you-go, requiring no sewing.
