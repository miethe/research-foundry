# Source Card: Amazon Bedrock now supports multi-agent collaboration

## Summary
This first-party AWS What's New announcement (March 10, 2025) marks the general availability of multi-agent collaboration for Amazon Bedrock, a supervisor-agent architecture for coordinating networks of specialized agents on complex, multi-step workflows. The GA release bundles four notable capabilities: Inline Agents (runtime role/behavior adjustment), Payload Referencing (linked-data references instead of embedded payloads), CloudFormation/CDK templating for reusable agent teams, and agent monitoring/observability. It is offered in all AWS Regions where Bedrock is supported.

## Key evidence
- (ev_001) Multi-agent collaboration for Amazon Bedrock reached GA on March 10, 2025, using a supervisor agent to coordinate specialized agents.
- (ev_002) The capability targets complex, multi-step workflows and scaling AI-driven applications.
- (ev_003) Inline Agents dynamically adjust agent roles and behaviors at runtime.
- (ev_004) Payload Referencing references linked data instead of embedding it per request, reducing data transfer and operational cost.
- (ev_005) CloudFormation and CDK support enables reusable, cross-account agent-team templates.
- (ev_006) Agent monitoring and observability features were added for tracking and optimizing agent interactions.
- (ev_007) Available in all AWS Regions where Amazon Bedrock is supported.
