# Source Card: LangGraph Multi-Agent Supervisor (langgraph-supervisor reference)

## Summary
The langgraph-supervisor package (LangChain, v0.0.31) provides a prebuilt central-supervisor pattern for multi-agent LangGraph systems. create_supervisor() builds a supervising LLM node that manages a list of worker agents (each a CompiledStateGraph, functional-API workflow, or Pregel object), and the handoff/agent_name helpers implement delegation to a worker, return of control, supervisor-mediated message forwarding, and agent-name tracking. All communication and task delegation flow through the central supervisor node rather than peer-to-peer between workers.

## Key evidence
- (ev_001) create_supervisor() is the primary function for building a central multi-agent supervisor.
- (ev_002) The supervisor manages a list of agents (CompiledStateGraph, functional-API workflow, or Pregel object).
- (ev_003) create_handoff_tool() creates a tool that hands off control to a named worker agent.
- (ev_004) create_handoff_back_messages() builds the message pair that returns control to the supervisor.
- (ev_005) create_forward_message_tool() forwards a worker message by name so the supervisor mediates communication and avoids information loss / token cost.
- (ev_006) with_agent_name() tags messages with agent names to keep multi-agent history coherent.
