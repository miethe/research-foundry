# Source Card: langgraph-swarm-py (LangGraph Multi-Agent Swarm)

## Summary
langgraph-swarm-py is LangChain's official Python library for building "swarm"-style multi-agent systems on LangGraph, where specialized agents dynamically hand off control to one another with no central coordinator. The library tracks an active agent in state so multi-turn conversations resume with the last-active agent, exposes create_handoff_tool() for customizable inter-agent handoffs and create_swarm() to assemble the workflow from a list of agents, and requires compiling with a checkpointer for multi-turn state. Latest release is 0.1.0 (Dec 4, 2025).

## Key evidence
- (ev_001) Agents dynamically hand off control based on specializations, with no central coordinator.
- (ev_002) The system remembers the last-active agent so multi-turn conversations resume with that agent.
- (ev_003) By default, swarm agents use handoff tools built with the prebuilt create_handoff_tool helper.
- (ev_004) Handoff tools allow customizing name/description and adding tool-call arguments (e.g. a task description for the next agent).
- (ev_005) create_swarm() builds the workflow from a list of agents and takes a default_active_agent argument.
- (ev_006) Multi-turn state requires compiling the swarm with a checkpointer.

- README does not benchmark performance or scale; claims are architectural/API descriptions, not empirical results. Version 0.1.0 indicates an early-stage release; APIs may change.
