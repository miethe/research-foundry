# Source Card: Launching the FutureHouse Platform: AI Agents for Scientific Discovery

## Summary
This FutureHouse launch announcement (May 1, 2025) introduces a free web + API platform shipping with four scientific AI agents: Crow (general-purpose literature search, API-oriented), Falcon (deep literature synthesis with access to specialized databases like OpenTargets), Owl (precedent search — "Has anyone done X before?"), and Phoenix (an experimental ChemCrow deployment for chemistry experiment planning). It establishes the product scope and FutureHouse's self-reported benchmark claims for the search agents.

## Key evidence
- (ev_001) Platform launched May 1, 2025, free via web interface and API.
- (ev_002) Crow: general-purpose literature-search agent for concise scholarly answers, designed for API use.
- (ev_003) Falcon: deep literature reviews with access to specialized databases like OpenTargets.
- (ev_004) Owl (formerly HasAnyone): precedent search — "Has anyone done X before?"
- (ev_005) Phoenix (experimental): ChemCrow deployment for planning chemistry experiments.
- (ev_006) Crow/Falcon/Owl benchmarked to outperform major frontier search models on retrieval precision and accuracy.
- (ev_007) Agents reported to have better precision than PhD-level researchers in head-to-head literature search.

## Limitations
- Performance/benchmark claims (ev_006, ev_007) are vendor self-reported in a first-party launch post; cited benchmarks [1][2][3] are not independently verified here.
- The page text does not mention "PaperQA2" in connection with Crow, even though some secondary summaries assert it; that linkage is not supported by this source and was omitted.
