# Source Card: Pricing - Claude API Docs

## Summary
This first-party Anthropic documentation page is the authoritative reference for Claude API pricing as of mid-2026. It enumerates per-model token rates (input, cache write/read, output), feature-specific pricing for server-side tools (web search, web fetch, code execution), and the two-dimension billing model (tokens plus session runtime) for Claude Managed Agents. It establishes concrete unit economics for an evidence-first research swarm that mixes cheap extraction models with expensive synthesis models and meters web-search usage.

## Key evidence
- (ev_001) Web search is $10 per 1,000 searches on the Claude API, plus standard token costs.
- (ev_002) Each search counts as one use regardless of result count; errored searches are not billed.
- (ev_003) Web fetch carries no additional charge beyond token costs for fetched content.
- (ev_004) Code execution is free when used with web search or web fetch.
- (ev_005) Sonnet 4.6 = $3/$15 per MTok (in/out), $0.30/MTok cache read; Haiku 4.5 = $1/$5 per MTok (cheapest tier); Opus 4.8 = $5/$25 per MTok.
- (ev_006) Managed Agents session runtime is $0.08 per session-hour, metered only while `running`.
- (ev_007) Managed Agents bills on two dimensions — tokens (standard Model pricing) and session runtime.

## Limitations
- Prices are point-in-time (page dated 2026-06) and Anthropic explicitly directs readers to claude.com/pricing for the most current figures; re-verify before asserting as current.
