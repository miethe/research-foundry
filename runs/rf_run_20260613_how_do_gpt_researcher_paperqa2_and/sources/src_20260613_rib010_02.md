# Source Card: PIP Package - GPT Researcher Documentation

## Summary
This is the official GPT Researcher documentation for its pip-installable Python package. It documents installation (`pip install gpt-researcher`, requiring Python 3.10+) and the accessor methods exposed on the researcher object. These methods surface research outputs and provenance: get_research_sources() (title, content, images), get_source_urls() (URLs used), get_research_context() (all retrieved info), get_costs() (tokens consumed), and get_research_images() (images found).

## Key evidence
- (ev_001) GPT Researcher is installed from PyPi via pip install gpt-researcher.
- (ev_002) The package requires Python 3.10 or newer.
- (ev_003) get_research_sources() retrieves a list of research sources including title, content, and images fields.
- (ev_004) get_research_context() returns all retrieved information, combining sources with corresponding content.
- (ev_005) get_costs() returns the number of tokens consumed during the research process.
- (ev_006) get_research_images() retrieves a list of images found during the research process.
- (ev_007) get_source_urls() returns the list of URLs used to gather information for the research.
