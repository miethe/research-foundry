# Source Card: assafelovic/gpt-researcher

## Summary
The official GitHub repository for GPT Researcher, an open-source autonomous deep-research agent that aggregates web and local sources to produce cited reports. The README documents its operating economics (Deep Research ~5 min, ~$0.40/run with o3-mini at high reasoning effort), an Apache 2-licensed self-hostable codebase, a Python API (GPTResearcher → conduct_research → write_report), LLM-agnostic configuration via OPENAI_BASE_URL, and hybrid Tavily + MCP retrieval connecting to GitHub repos, databases, and custom APIs.

## Key evidence
- (ev_001) GPT Researcher aggregates more than 20 sources per research run to form objective conclusions.
- (ev_002) Deep Research mode takes about 5 minutes and costs about $0.40 per run using o3-mini at high reasoning effort.
- (ev_003) The project is distributed under the Apache 2 license, supporting self-hosted use.
- (ev_004) The Python API instantiates GPTResearcher with a query, then awaits conduct_research() and write_report().
- (ev_005) MCP integration connects specialized data sources such as GitHub repos, databases, and custom APIs.
- (ev_006) The retriever supports a hybrid of web search (Tavily) and MCP retrievers via the RETRIEVER variable.
- (ev_007) LLM-agnostic configuration targets custom OpenAI-compatible APIs via OPENAI_BASE_URL.

## Limitations
- Cost and time figures (~5 min, ~$0.40/run) are maintainer-reported in the README, not independently benchmarked, and depend on the specific model (o3-mini, high reasoning effort) and run configuration.
- README framing is promotional in places (e.g., "first open deep research agent") and should be treated as a vendor self-description.
