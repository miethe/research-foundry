# Source Card: Core: Robust Factual Precision with Informative Sub-Claim Identification

## Summary
This arXiv paper (Jiang et al., JHU; v2 Oct 2024) shows that Decompose-Then-Verify factual-precision metrics such as FActScore can be gamed by padding generations with obvious or repetitive subclaims, artificially inflating scores. It introduces Core, a plug-and-play subclaim-selection component that formulates subclaim subselection as a constrained optimization problem to keep only unique and informative subclaims, making the resulting metrics substantially more robust across knowledge domains. The authors release a modular evaluation framework and an expanded FActScore biography dataset.

## Key evidence
- (ev_001) Decompose-Then-Verify metrics like FActScore can be manipulated to inflate scores via obvious or repetitive subclaims.
- (ev_002) Core is a customizable, plug-and-play subclaim selector filtering by uniqueness and informativeness.
- (ev_003) Core formalizes subclaim selection as a constrained optimization problem.
- (ev_004) Core-augmented metrics are substantially more robust across many knowledge domains.
- (ev_005) Core is a drop-in replacement for the decomposition component of existing FP pipelines.
- (ev_006) Authors release a modular evaluation framework plus an expanded FActScore biography dataset.
