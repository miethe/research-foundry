# Source Card: google-research-datasets/Attributed-QA (Attributed Question Answering dataset)

## Summary
The Attributed-QA repository from Google Research releases human-rated system outputs for the Attributed Question Answering (AQA) task, in which each answer must be paired with an attribution — a pointer into a fixed Wikipedia corpus that supplies supporting evidence and satisfies the AIS (Attributable to Identified Sources) conditions. The release (83,030 instances across 3,610 examples x 23 systems, over a 2021-10-13 Wikipedia snapshot) is positioned to enable reproducible evaluation of attributed LLMs for information-seeking. It accompanies arXiv:2212.08037, which uses human attribution annotations as a gold standard and shows that a correlated automatic metric (AutoAIS) is suitable for development.

## Key evidence
- (ev_001) Attributed QA is framed as a key first step toward attributed LLMs, releasing human-rated system outputs.
- (ev_002) The task maps a question input to an (answer, attribution) output pair.
- (ev_003) Attribution is a pointer into a fixed underlying corpus (claim-to-source linking).
- (ev_004) Attribution must give supporting evidence and satisfy AIS conditions.
- (ev_005) 83,030 instances (3,610 examples x 23 systems) over a 2021-10-13 Wikipedia snapshot.
- (ev_006) The accompanying paper treats human annotations as gold and shows a correlated automatic metric is suitable for development.
