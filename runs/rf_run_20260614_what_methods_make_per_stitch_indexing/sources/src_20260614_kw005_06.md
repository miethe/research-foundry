# Source Card: Wooly Graphs: A Mathematical Framework for Knitting

## Summary
This arXiv preprint develops a graph-theoretic foundation for modeling knitting, defining knit objects with a knot-theoretic component and proposing three models for any arbitrary knit object: a simple undirected graph, a simple directed graph, and a directed multigraph. It introduces complexity-based categories of knitting structures and proves that deciding whether a knit object of a given class exists for a given graph is NP-hard in general, while special cases tied to common knitting techniques admit linear- and polynomial-time algorithms. The directed formalism (with edge direction encoding yarn-path order) is the formal analog of stable per-stitch identity.

## Key evidence
- (ev_001) Three graph models for any knit object — simple undirected graph, simple directed graph, directed multigraph — each with a knot-theoretic component.
- (ev_002) Natural categories of knitting structures defined by their complexity.
- (ev_003) Existence of a knit object for a given graph is NP-hard in general; specific cases admit linear- and polynomial-time algorithms.
- (ev_004) The framework supports both analyzing knit objects from graphs and generating knit objects from graphs (bidirectional).
- (ev_005) Authorship and arXiv provenance (2407.00511, submitted 29 Jun 2024).
