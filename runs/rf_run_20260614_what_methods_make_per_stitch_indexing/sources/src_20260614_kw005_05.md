# Source Card: CrochetPARADE (Crochet PAttern Renderer, Analyzer, and DEbugger) — Repository README

## Summary
CrochetPARADE is an open-source (GPLv3, client-side computation) platform by Svetlin Tassev that uses a custom formal grammar to define, validate, and render 2D and 3D crochet patterns. Its SVG export labels each stitch by type, row number, and position within a row—mirroring standard crochet diagrams—and its 3D viewer supports animation, stitch highlight/hide, yarn thickness/color changes, and on-hover per-stitch info. The project is dual-hosted (primary on Codeberg, mirror on GitHub), with the user manual under CC BY-NC-SA and showcase patterns in the public domain.

## Key evidence
- (ev_001) Website and all computational components are free/open source under GPLv3, free in perpetuity.
- (ev_002) Showcase patterns are public domain; manual is CC BY-NC-SA; grammar is uncopyrightable/public domain.
- (ev_003) SVG export identifies each stitch by type, row number, and position within a row, matching standard crochet diagrams.
- (ev_004) 3D viewer supports animation, stitch highlight/hide, yarn thickness/color changes, and hover-for-stitch-info.
- (ev_005) All calculations run locally with no central-server collection; large patterns can take minutes.
- (ev_006) Primary repo on Codeberg with a GitHub mirror; live platform at crochetparade.org.
- (ev_007) Custom grammar parses and checks patterns for correctness, then builds a 3D model, avoiding plain-English ambiguity.
