# Source Card: SceneView/sceneform-android — Sceneform Maintained (ARCore + Filament)

## Summary
This is the official repository README for "Sceneform Maintained," an ARCore Android SDK that uses Google Filament as its 3D engine and is positioned as the continuation of Google's archived Sceneform. It loads glTF/GLB models natively (from assets, res/raw, local file, or http/https URL) with animation support, but the maintainers explicitly mark it as no longer updated and steer users toward the Kotlin SceneView successor. Two successor lineages exist — the Java continuation (this repo, latest 1.23.0 on 2023-08-04) and the Kotlin SceneView Composable — both on Filament + ARCore.

## Key evidence
- (ev_001) Sceneform Maintained is an ARCore Android SDK using Google Filament as the 3D engine, continuing the archived Sceneform.
- (ev_002) Native glTF/GLB ingestion from assets, res/raw, local file, or http/https URL, replacing sfa/sfb/fbx/obj.
- (ev_003) Maintainers state the framework is no longer updated and recommend SceneView instead.
- (ev_004) Two documented successor lineages: Java continuation (this repo) and Kotlin SceneView.
- (ev_005) Latest release is 1.23.0 (2023-08-04); no newer releases.
- (ev_006) Tracks latest ARCore SDK and Filament; supports model animations.

## Limitations
- Self-described, primary-source maintenance status: the README itself states the framework "isn't updated anymore," and the latest release (1.23.0) dates to 2023-08-04, so capability claims reflect a maturity/abandonment-risk state rather than an actively maintained library.
