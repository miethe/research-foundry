# Source Card: CrochetPARADE README (GitHub)

## Summary
CrochetPARADE (Crochet PAttern Renderer, Analyzer, and DEbugger) is a browser-based platform by Svetlin Tassev that lets users author crochet patterns in a custom language grammar, parses and checks them for correctness, and renders a virtual 3D model. It adds analysis (flagging loose/tight stitches to reduce blocking), an interactive 3D viewer, and export to crochet charts, labeled SVGs, and Blender-importable 3D files. It is built on SVG.js and three.js.

## Key evidence
- (ev_001) Custom grammar defines stitches/patterns; parser checks correctness, then builds a virtual model rendered in 3D.
- (ev_002) Grammar aims for accuracy/precision, avoiding plain-English ambiguity.
- (ev_003) Debugging flags loose/tight stitches to fix before crocheting, reducing the need for blocking.
- (ev_004) Interactive 3D view: rotate/zoom/pan, animate creation, highlight/hide stitches, change yarn thickness and color.
- (ev_005) Export: auto-generated crochet chart in standard symbols, plus SVG labeling stitches by type/row/position.
- (ev_006) Built on SVG.js and three.js; exports 3D files importable into Blender.
