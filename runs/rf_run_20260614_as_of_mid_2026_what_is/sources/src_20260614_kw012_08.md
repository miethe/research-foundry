# Source Card: CrochetPARADE (Crochet PAttern Renderer, Analyzer, and DEbugger)

## Summary
CrochetPARADE is a free, open-source (GPLv3) browser-based platform for creating, visualizing, and analyzing both 2D and 3D crochet patterns, with a custom stitch grammar that renders to virtual 3D models. All computation runs locally in the browser with no server transmission; the platform targets desktop browsers (Chrome, Brave, Firefox) and is not designed for mobile. As of its Feb 13, 2026 update, it supports exporting instructions text, a Standard Crochet Chart SVG, and a GLTF 3D model (Blender-compatible), and recently added a 3D-model-to-instructions Remesher and an STL export for 3D printing.
