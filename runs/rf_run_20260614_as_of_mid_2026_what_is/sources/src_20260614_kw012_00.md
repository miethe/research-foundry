# Source Card: Ravelry: Apps that connect to Ravelry (official directory)

## Summary
Ravelry's own "/about/apps" page explicitly states there is no official first-party Ravelry app and instead presents a directory of third-party apps "created by Ravelers and other developers." The directory spans row counters / project trackers (knitCompanion, Row Counter, YarnBuddy, Pocket Knitting/Crochet, Loopsy), pattern discovery/management (kntd:discover, Pattrick, Ravelgurumi, Ravit/Ravit 2, Yarn Squirrel), a pattern-design tool (Stitch Fiddle), and stash managers (Yarn Stasher, YarnCat, YarnBuddy). The page confirms the entire Ravelry mobile ecosystem is third-party, supporting the thesis that no integrated official runtime exists.

## Key evidence
- (ev_001) Ravelry confirms there is no official Ravelry app; the page is a directory of third-party apps built by Ravelers and other developers.
- (ev_002) knitCompanion is a listed pattern-tracking connector for knitters/fiber artists.
- (ev_003) Several third-party row-counter / project-tracking connectors exist (Row Counter, YarnBuddy, Pocket Knitting, Pocket Crochet, Loopsy).
- (ev_004) Pattern discovery/management connectors include kntd:discover, Pattrick, Ravelgurumi (amigurumi), and Ravit/Ravit 2.
- (ev_005) Yarn Squirrel brings the full Ravelry library to iOS and stores PDF patterns alongside it.
- (ev_006) Stitch Fiddle is a connected pattern-design tool for knitting, cross stitch, and crochet.
- (ev_007) Stash management is served by third-party connectors including Yarn Stasher (AI-powered Smart Scan), YarnCat, and YarnBuddy.
