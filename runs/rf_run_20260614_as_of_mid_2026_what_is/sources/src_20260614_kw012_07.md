# Source Card: CrochetPARADE (Crochet PAttern Renderer, Analyzer, and DEbugger) - GitHub repo

## Summary
CrochetPARADE is an open-source platform (web app at crochetparade.org, source on GitHub) that lets users author crochet patterns in a custom, ambiguity-free grammar and render them as both 2D charts and interactive 3D models. It adds a "debug" loop that flags overly loose/tight stitches before crocheting, supports interactive 3D manipulation (rotate/zoom/pan, animation, stitch highlight/hide, yarn thickness/color), and exports standard crochet charts, annotated SVGs, and Blender-importable 3D files. All computation runs locally on-device with no server transmission; the software is GPLv3 and the grammar is public domain.

## Key evidence
- (ev_001) CrochetPARADE creates, visualizes, and analyzes both 2D and 3D crochet patterns via a custom stitch-pattern grammar.
- (ev_002) The grammar is designed to remove plain-English instruction ambiguity for accuracy and precision.
- (ev_003) A debug feature flags overly loose/tight stitches so users can fix patterns before crocheting, reducing blocking.
- (ev_004) Interactive 3D view supports rotate/zoom/pan, animation, stitch highlight/hide, and yarn thickness/color changes.
- (ev_005) Exports include standard-symbol crochet charts, SVGs with stitch metadata, and Blender-importable 3D files.
- (ev_006) All calculations run locally on-device with no central-server collection or internet transmission.
- (ev_007) Software is GPLv3; the grammar is public domain; project hosted at crochetparade.org.
