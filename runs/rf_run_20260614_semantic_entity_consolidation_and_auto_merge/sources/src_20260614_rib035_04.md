# Source Card: Entity Matching using Large Language Models

## Summary
Peeters, Steiner & Bizer (EDBT 2025) study generative LLMs as alternatives to fine-tuned pre-trained language models (PLMs like BERT/RoBERTa/Ditto) for entity matching — deciding whether two entity descriptions refer to the same real-world entity. They show that strong LLMs such as GPT-4 reach or exceed fine-tuned PLM accuracy in zero-shot settings without task-specific training data, are markedly more robust to unseen entities, and require per-model/per-dataset prompt tuning rather than a single universal prompt.

## Key evidence
- (ev_001) The best LLMs need no or only a few examples to match PLMs fine-tuned on thousands, and are more robust to unseen entities.
- (ev_002) GPT-4 zero-shot F1: Abt-Buy 95.78, Walmart-Amazon 89.67, Amazon-Google 76.38, WDC Products 89.61, DBLP-Scholar 89.82, DBLP-ACM 98.41.
- (ev_003) Zero-shot GPT-4 reaches comparable or better results than PLMs fine-tuned on thousands of training pairs.
- (ev_004) Fine-tuned baselines: RoBERTa 91.21 / Ditto 91.31 (Abt-Buy), RoBERTa 87.02 / Ditto 86.39 (Walmart-Amazon); GPT-4 zero-shot beats these on the product datasets.
- (ev_005) No single best prompt exists; prompts must be tuned per model/dataset combination.
- (ev_006) On unseen entities, PLMs drop 22-61% F1 while GPT-4 outperforms the best transferred PLM by 34% F1.

## Limitations
- GPT-3.5-Turbo zero-shot F1 is reported across multiple API snapshots (Turbo03/Turbo06) rather than a single value, so cited GPT-3.5 numbers depend on the snapshot; hosted-LLM results are subject to model drift across API versions.
