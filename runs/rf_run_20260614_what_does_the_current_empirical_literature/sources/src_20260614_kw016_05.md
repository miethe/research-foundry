# Source Card: Translation of User Crochet Patterns to CrochetPARADE Syntax Using Large Language Models

## Summary
This peer-reviewed AAAI Symposium Series paper (Dias & Karim, Heriot-Watt University, 2025) investigates using LLMs to translate user-generated crochet patterns into the CrochetPARADE domain-specific syntax. The authors built the first structured, open-source collection of crochet patterns paired with CrochetPARADE translations for machine-learning use, and evaluated baseline, few-shot, and fine-tuning approaches. Their best result, fine-tuning DeepSeek-R1-Distill-Llama8b, reached 74% accuracy, positioned as a path to make structured, validatable pattern creation accessible to non-expert crocheters.

## Key evidence
- (ev_001) Created the first structured, open-source crochet-pattern collection (patterns paired with CrochetPARADE translations) built for ML applications.
- (ev_002) Evaluated baseline, few-shot, and fine-tuning LLM approaches for pattern-to-DSL translation.
- (ev_003) Best result: fine-tuned DeepSeek-R1-Distill-Llama8b reached 74% accuracy.
- (ev_004) Frames LLM translation as bridging standard notation and CrochetPARADE's non-intuitive syntax to aid novices.
- (ev_005) Motivates the work via the spatial-reasoning / structural-stitch barrier novices face.
- (ev_006) Provenance: peer-reviewed AAAI Symposium Series Vol.6 No.1, pp.200-208, 2025-08-01, DOI 10.1609/aaaiss.v6i1.36054, Human-AI Collaboration track.
