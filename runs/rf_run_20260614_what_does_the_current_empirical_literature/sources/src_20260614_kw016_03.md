# Source Card: Modeling Crochet Patterns with a Force-directed Graph Layout

## Summary
This peer-reviewed Eurographics 2025 (Expressive/WICED) paper by Émile Greer and David Mould (Carleton University) presents a forward engine that converts a written amigurumi crochet pattern (row-by-stitch notation) into a stitch graph and then computes a force-directed graph layout to produce a 3D model accurate in shape and size to the hand-crocheted result. Stitches are nodes; sequential and working connections are edges; the layout minimizes edge-length and curvature energies (adapting Isenburg et al.) via the Ceres Solver, modeling stuffing as graph inflation. Source code is released, and the authors envision an iterative/incremental design workflow where designers preview and adjust patterns digitally instead of crocheting and unraveling repeatedly.

## Key evidence
- (ev_001) Written pattern -> graph -> force-directed layout produces a 3D model matching the hand-crocheted piece in shape and size.
- (ev_002) Designers can preview/adjust digitally without physically crocheting; oriented to amigurumi, extensible to clothing.
- (ev_003) Stitch-graph encoding: nodes are stitches; horizontal edges are sequential connections, vertical edges are working connections.
- (ev_004) Layout adapts Isenburg et al. as a non-linear least-squares minimization of edge-length and curvature energies (stuffing == inflation).
- (ev_005) Implemented with Ceres Solver; static λ = 0.65 (λ = 0.9 for bunny head/body); linear solver is the cost bottleneck.
- (ev_006) Performance: small pieces converge in seconds, but ~700-800-stitch pieces failed to converge after 2+ minutes.
- (ev_007) Open-source reference implementation released at github.com/EnbyMonkey/modelingamigurumi.

## Limitations
- Tested only on simple, beginner-level patterns and single-crochet-like stitches; explicitly too narrow for intermediate/advanced patterns.
- Assumes a uniform edge length of 1, which the authors acknowledge is inaccurate in general (real stitch sizes vary with tension, hook, yarn weight, stitch type).
- Piece joining and certain shaping effects are specified ad hoc / by hand; force-directed layouts are slow and large patterns may not converge.
