# Source Card: CrochetPARADE (Crochet PAttern Renderer, Analyzer, and DEbugger) repository

## Summary
CrochetPARADE is an open-source platform (author: Svetlin Tassev) that parses crochet patterns written in a custom DSL grammar, checks them for correctness, and deterministically builds a 3D virtual model that reproduces the pattern exactly as its author intended — with no physics solver or convergence step. All computation runs locally on-device, which keeps data private but makes the tool sluggish on old hardware and pushes large patterns (tens of thousands of stitches) to minutes-or-more render times. The platform offers interactive 3D inspection (rotate/zoom/pan, animation, highlight/hide, yarn color/thickness, hover info) and exports to standard-symbol charts, labeled SVGs, and Blender-importable 3D files, all under the GPLv3 copyleft license.

## Key evidence
- (ev_001) Deterministic parse → correctness-check → 3D virtual model build pipeline.
- (ev_002) Shared pattern text renders exactly as intended by the author, free of ambiguity.
- (ev_003) Local on-device computation; sluggish on old hardware; (tens of) thousands of stitches take minutes or more.
- (ev_004) Interactive features: rotate/zoom/pan, animation, highlight/hide, yarn thickness/color, hover-to-read stitch info.
- (ev_005) Exports: standard-symbol crochet chart, SVG with labeled stitch connections, Blender-importable 3D files.
- (ev_006) Released under GPLv3 (copyleft) — a reuse constraint for closed-source apps.

## Limitations
- Self-descriptive primary source (the project's own README); performance numbers are author-stated, not independently benchmarked.
- The "6,085 patterns encoded into the CrochetPARADE grammar" claim (CrochetBench validation at scale) comes from a separate arXiv paper (arXiv:2511.09483), NOT from this repo's README; it is therefore intentionally omitted from the verbatim evidence here and should be cited to its own source if used.
