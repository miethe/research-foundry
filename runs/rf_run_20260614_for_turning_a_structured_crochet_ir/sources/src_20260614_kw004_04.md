# Source Card: CrochetPARADE: Crochet PAttern Renderer, Analyzer and DEbugger

## Summary
CrochetPARADE is a free, GPLv3, browser-based platform by Svetlin Tassev (2023, page updated Feb 13 2026) that lets users author crochet patterns in a custom domain-specific grammar, then parses/validates those patterns for correctness and renders an interactive 3D virtual model. It serves as a renderer, analyzer, and debugger: it flags overly loose or tight stitches to reduce blocking, exports standard crochet charts/SVGs, and exports 3D files importable into Blender. All computation runs locally with no server transmission.

## Key evidence
- (ev_001) Custom language grammar defines stitches/patterns to remove plain-English ambiguity.
- (ev_002) Code parses and checks any user-provided pattern for correctness before rendering a 3D model.
- (ev_003) Identifies overly loose/tight stitches so users can swap them before crocheting, reducing blocking.
- (ev_004) Interactive 3D features: rotate/zoom/pan, animate creation, highlight/hide stitches, change yarn thickness and color.
- (ev_005) Exports 3D files importable into Blender for further manipulation.
- (ev_006) Authored by Svetlin Tassev (2023); free and open source under GPLv3.
- (ev_007) All calculations run locally on-device with no data sent to a server (can be slow on old hardware / large patterns).

## Limitations
- Source is the project's own landing page (self-description / marketing), not an independent evaluation.
- The page does not state Codeberg/GitHub source mirrors, so that claim is not supported by this source.
- Performance caveat: large patterns (tens of thousands of stitches) can take minutes or more to calculate; not designed for mobile.
