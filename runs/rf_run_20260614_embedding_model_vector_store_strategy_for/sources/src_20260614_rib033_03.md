# Source Card: sentence-transformers/all-MiniLM-L6-v2 - Hugging Face model card

## Summary
The official Hugging Face model card for `sentence-transformers/all-MiniLM-L6-v2`, a compact sentence-embedding model that maps text to a 384-dimensional dense vector space for clustering and semantic search. It was fine-tuned from the pretrained `nreimers/MiniLM-L6-H384-uncased` base model using a contrastive learning objective over a concatenation of datasets totaling more than 1 billion sentence pairs. The card documents its small size (22.7M parameters), 256 word-piece input truncation, and self-reported retrieval benchmark results.

## Key evidence
- (ev_001) The model maps sentences and paragraphs to a 384-dimensional dense vector space, suitable for clustering and semantic search.
- (ev_002) The model is intended as a sentence and short-paragraph encoder, and by default truncates input longer than 256 word pieces.
- (ev_003) Training uses a contrastive learning objective predicting which randomly sampled sentence was actually paired with a given sentence.
- (ev_004) The model was fine-tuned on a concatenation of multiple datasets totaling over 1 billion sentence pairs (1,170,060,424 training tuples).
- (ev_005) The model is fine-tuned from the pretrained nreimers/MiniLM-L6-H384-uncased base model.
- (ev_006) The model has 22.7M parameters and reports 50.17 on the ArguAna retrieval task.

## Limitations
- Metrics and training details are self-reported by the model authors and not independently audited; benchmark numbers reflect the authors' specific evaluation configuration. The card was published circa 2021-08, so it predates many newer embedding models.
