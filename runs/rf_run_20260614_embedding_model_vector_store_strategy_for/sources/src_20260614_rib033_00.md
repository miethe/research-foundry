# Source Card: BAAI/bge-m3 - Hugging Face model card

## Summary
BGE-M3 is an open multilingual embedding model published by the Beijing Academy of Artificial Intelligence (BAAI). Its defining feature ("M3") is versatility across Multi-Functionality, Multi-Linguality, and Multi-Granularity: a single model performs dense, sparse (lexical), and multi-vector (ColBERT) retrieval, supports 100+ languages, and handles inputs from short sentences up to 8192-token documents with a 1024-dimension dense embedding. For RAG, the authors recommend a hybrid retrieval (dense + sparse) pipeline followed by cross-encoder re-ranking for the highest accuracy.

## Key evidence
- (ev_001) BGE-M3 produces dense embeddings of dimension 1024 and supports a maximum input sequence length of 8192 tokens.
- (ev_002) BGE-M3 can simultaneously perform three retrieval functionalities: dense, multi-vector (ColBERT), and sparse (lexical) retrieval.
- (ev_003) BGE-M3 supports more than 100 working languages.
- (ev_004) BGE-M3 can process inputs from short sentences up to long documents of up to 8192 tokens.
- (ev_005) The model card recommends a RAG pipeline of hybrid retrieval (dense + sparse) followed by re-ranking.
- (ev_006) Re-ranking with a cross-encoder is recommended because it achieves higher accuracy than a bi-encoder embedding model.

## Limitations
- This is the authors' own model card (primary but promotional); benchmark/accuracy claims reflect the publisher's framing rather than independent evaluation. Spec values are as of the 2024-02 model release and may differ from later checkpoints or community forks.
