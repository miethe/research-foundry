# Source Card: MMTEB: Massive Multilingual Text Embedding Benchmark

## Summary
MMTEB is a community-driven, large-scale expansion of the MTEB embedding benchmark, spanning over 500 quality-controlled tasks across 250+ languages and adding novel categories (instruction following, long-document retrieval, code retrieval). It establishes that a 560M-parameter model (multilingual-e5-large-instruct) is the best publicly available performer overall, beating much larger billion-parameter LLMs, and contributes compute-saving techniques (inter-task-correlation downsampling, hard-negative sampling) that preserve model rankings at a fraction of the cost. Accepted at ICLR 2025.

## Key evidence
- (ev_001) MMTEB expands MTEB to 500+ quality-controlled tasks across 250+ languages.
- (ev_002) Adds novel task categories: instruction following, long-document retrieval, code retrieval.
- (ev_003) Best public model is multilingual-e5-large-instruct (560M params), beating larger LLMs overall.
- (ev_004) Inter-task-correlation downsampling cuts compute while preserving relative model rankings.
- (ev_005) Hard-negative sampling creates smaller but effective retrieval splits, cutting compute.
- (ev_006) Zero-shot English benchmark keeps similar ranking order at a fraction of the cost.
- (ev_007) Submitted to arXiv 19 Feb 2025; accepted at ICLR 2025.

## Limitations
- Extraction is abstract-level; full methodology, exact task counts per category, and leaderboard tables were not read in detail.
- Rankings reflect publicly available models around early 2025 and may shift as newer embedding models ship.
