# Source Card: OpenAI text-embedding-3-small model page (pricing)

## Summary
This OpenAI developer documentation page establishes the per-token API pricing for the `text-embedding-3-small` embedding model: $0.02 per 1M tokens on both the standard and Batch API. It positions the model as an improved, more performant successor to the legacy `ada` embedding model, and its companion page prices the larger `text-embedding-3-large` model at $0.13 per 1M tokens — a roughly 6.5x difference relevant to embedding-cost tradeoffs.

## Key evidence
- (ev_001) text-embedding-3-small is priced at $0.02 per 1M tokens on the standard API.
- (ev_002) The Batch API rate is the same $0.02 per 1M tokens.
- (ev_003) Described as an improved, more performant successor to the ada embedding model.
- (ev_004) Companion model text-embedding-3-large is priced at $0.13 per 1M tokens.
- (ev_005) Listed embedding use cases: search, clustering, recommendations, anomaly detection, and classification.

## Limitations
- Pricing pages change over time; the $0.02 and $0.13 rates are as accessed on 2026-06-14 and should be re-verified for current figures.
- Context length and output-dimension specifications were not present in the fetched page content.
- The comparison to $0-per-token local open-weight models (bge-m3, MiniLM, nomic) is an analyst inference, not a statement made on this OpenAI page.
