# Source Card: M3-Embedding: Multi-Linguality, Multi-Functionality, Multi-Granularity Text Embeddings Through Self-Knowledge Distillation

## Summary
M3-Embedding (released as BGE-M3 by the BAAI BGE team) is a text embedding model that unifies dense, sparse (lexical), and multi-vector retrieval in a single model, supports more than 100 languages, and handles inputs up to 8,192 tokens for long-document retrieval. It is trained with a novel self-knowledge distillation method that fuses relevance scores across the three retrieval functionalities as a teacher signal, plus an optimized batching strategy for large batch sizes. The authors report state-of-the-art results on multilingual, cross-lingual, and long-document retrieval benchmarks.

## Key evidence
- (ev_001) BGE-M3 is versatile across multi-linguality, multi-functionality, and multi-granularity, covering 100+ working languages.
- (ev_002) One model simultaneously supports dense, multi-vector, and sparse retrieval.
- (ev_003) Handles inputs from short sentences to 8,192-token long documents.
- (ev_004) Trained via novel self-knowledge distillation using cross-functionality relevance scores as the teacher signal.
- (ev_005) Optimized batching strategy enables large batch size and high training throughput.
- (ev_006) First submitted 5 Feb 2024 (v1); latest revision v5 dated 12 Dec 2025, within active-maintenance freshness window.
